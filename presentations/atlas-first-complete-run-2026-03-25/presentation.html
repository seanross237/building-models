Architecture

5 Agents, 3 Levels

Hierarchical system where each level owns a different scope of decisions

Missionary
Owns the mission. Writes strategies, evaluates results.
Strategizer
Owns one strategy. Runs 10-16 explorations. Stop-hook loop.
Explorer
Researches one question
Librarian
Searches the knowledge base
Curator
Processes findings into library
OUTER LOOP
Missionary writes strategy, launches strategizer, evaluates final report, decides whether to create next strategy.
INNER LOOP
Strategizer queries librarian, designs exploration goal, launches explorer in tmux, processes report, launches curator. Repeat.

↓

↓

↓
Curator — fire-and-forget in tmux, processes report into library, checks duplicates
STOP HOOK
Shell script registered in ~/.claude/settings.json. When strategizer's context runs out, the hook matches the transcript path to the strategy directory and feeds a meta-prompt back in. Keeps the loop alive across context resets.
MONITORING
Strategizer polls explorer's REPORT.md line count. Can nudge after 5 min of no progress, kill after 15. Partial results survive on disk. The single manual step: approving the settings.json permission prompt.

The Derivation Chain

Spectral dimension ds = 4 → 2 as constructive axiom
Observed across 7+ independent QG programs. No one had used this as a starting point before.
No-go theorem: ghost-free theories cannot produce ds = 2
Hadamard factorization + Kallen-Lehmann representation. Ghost-free nonlocal theories give ds → 0.
Unique action: quadratic gravity (Stelle 1977)
ds = 2 forces 4-derivative terms. Diffeo invariance leaves only R2 and C2. Two dimensionless couplings.
Ghost resolved via fakeon quantization (Anselmi-Piva 2018)
Massive spin-2 mode removed from physical spectrum. Theory is both renormalizable AND unitary.
NOVEL
The derivation path (spectral dimension → no-go → unique action → fakeon) is new. The theory itself is Anselmi's existing program (25+ papers since 2017).
KEY FINDING
QG+F is the perturbative face of asymptotic safety. Both produce identical UV propagators (~1/p4). Same theory, different regimes — analogous to perturbative vs. non-perturbative QCD.

System Performance

What Worked, What Broke

3 test runs before the full mission. Each fixed real problems.

Worked
Stop hook loop kept the strategizer alive across context resets. Transcript path matching was simple and reliable after the initial bug was fixed.
Curator maintained a 95+ entry library with INDEX.md hierarchy. Deduplication, conflict checking, and hierarchical indexing worked. Root INDEX.md was missed once (prompt strengthened).
Librarian queries improved strategizer decision quality. Context from previous explorations prevented redundant work and enabled the strategizer to build on findings.
Context efficiency was excellent. Strategizer at 12% after 7 explorations. Could easily handle 16+.
Fixed Mid-Run
Run 2: Foreground explorers. Run 1 used foreground sub-agents that couldn't be monitored or timed out. Switched to tmux-based explorers with progress polling.
Run 2: Curator automation. Initially manual. Added fire-and-forget tmux launch after each exploration.
Run 2: Incremental writing. Explorer wrote 8 lines early, then nothing for 10 min, then 521 at end. Improved to 33 lines early by run 3.
Known Limitations
Settings.json permission prompt — requires one manual "2" approval per run. Could be pre-registered.
Internet outage recovery — strategizer got stuck on ConnectionRefused for 33 min. Recovered when nudged, but no auto-retry.
Single-prediction problem — methodology finding: all non-CMB signatures are 15+ orders of magnitude out of reach. The theory is effectively a one-test theory (r).

THE RESULT
QG+F is the unique unitary, Lorentz-invariant, diffeomorphism-invariant, local, perturbatively renormalizable quantum gravity in 4D. Eight independent derivation paths converge on it. No alternative exists within standard axioms.
The theory is not new (Anselmi et al.). The proof of its uniqueness, the spectral dimension derivation, and the no-go theorem are.

Why No Strategy 3

The uniqueness theorem closes the space. A third strategy would either reach the same conclusion within standard axioms, or abandon axioms into territory that is experimentally excluded (Lorentz violation) or unconstrained (non-perturbative, non-QFT). The remaining open questions — cosmological constant, non-perturbative completion, SM origin — are the frontier of all theoretical physics.

Unsolved
