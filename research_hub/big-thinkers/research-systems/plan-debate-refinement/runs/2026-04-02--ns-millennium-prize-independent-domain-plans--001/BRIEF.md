# Brief: Independent Cross-Domain Navier-Stokes Prize Plans

## Goal

Generate multiple genuinely independent research plans for trying to solve the
3D incompressible Navier-Stokes Millennium Prize problem.

This run is different from the earlier critique-based runs:

- do not critique one shared draft
- do not inherit a prior route ranking
- do not merely refine an existing plan

Instead, each planner should produce its own plan from scratch using only the
authoritative current-status document plus its assigned domain lens.

## Authoritative Source

- `current-status-of-navier-stokes.md`

## Requirements For Each Domain Planner

Each planner must produce a full plan, not just comments.

Each plan must include:

- core judgment about where the real live frontier is
- main theorem target
- fallback theorem target
- route ranking or route structure
- first 2-3 concrete theorem steps
- hard stop conditions
- what would count as genuine prize-facing progress
- what the planner refuses to spend time on because the status file has already
  closed it

## Shared Constraints

- treat `current-status-of-navier-stokes.md` as authoritative for current local
  status
- do not reopen routes the status file marks structurally closed unless the plan
  names a genuinely new ingredient that escapes the stated closure
- tie active work to accepted PDE theorem targets or robust no-go theorems
- be explicit when a route is a bounded prerequisite/no-go campaign rather than a
  real mainline

## Domain Lenses To Cover

1. critical spaces / compactness-rigidity
2. harmonic analysis / pressure / nonlinear estimates
3. exact helical triad dynamics / shell-crossing mechanisms
4. geometric depletion / vorticity-direction regularity
5. proof architecture / no-go theorems / theorem-object engineering

## Synthesis Goal

After the five independent plans are produced, they will be compared and merged
into one final synthesis that says:

- where the plans agree
- where they fundamentally disagree
- what the best merged plan is
- whether any prize-facing main route is actually justified right now
