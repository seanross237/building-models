Navier-Stokes regularity problem at Millennium-Prize level, using the current
authoritative status picture rather than reopening already-closed routes.

## Operating Constraints

- Treat `current-status-of-navier-stokes.md` as authoritative.
- Do not spend primary effort on routes already marked structurally closed.
- Optimize for theorem-bearing progress, not attractive speculation.
- Prefer exact-NS object-definition progress over another round of loose

  harmless-refinement stability, and exact closure bookkeeping.

## Current Frontier From The Status File

The status file says the classical estimate routes are largely exhausted:

- enstrophy regularity loops back into BKM;
- Vasseur / De Giorgi pressure improvement is capped at `beta = 4/3`;
- epsilon-regularity is structurally capped;
- the `H^1` pressure route and harmonic far-field repair are dead;
- Tao-adjacent work has sharpened the object-definition problem, but not yet
  produced a firewall theorem.

The live frontier is narrow:

1. exact packet-family theorem on the frozen near-circuit ledger
2. leakage firewall lemma on the same ledger
3. intrinsic phase/coherence route on shell-crossing exact triads
4. conditional critical-element bridge, only if a canonical extraction lemma
   appears
5. helical-sign route as support evidence, not main theorem object

There is also a separate non-Tao frontier:

- alignment + vorticity-direction regularity
- direct pressure-Hessian analysis
- vorticity-direction Lipschitz regularity
