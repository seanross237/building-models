# Downstream Object Freezes Should Attach One Next Question, One Invariant Gate, One Carried Witness, And One Named Debt

When a step promotes repaired or shortlisted objects into a downstream freeze,
each promoted object should leave the step with four concrete handoff items:

- one exact next question;
- one invariant gate or admissibility sheet on the carried ledger;
- one smallest carried-forward witness family or stress set;
- and one named same-currency debt that still blocks theorem or
  counterexample success.

That package keeps the freeze operational.
It prevents later steps from inheriting labels such as `survivor`,
`repaired object`, or `promoted gate` without knowing what exact theorem or
counterexample test is actually next, what observable remains fixed, and what
missing proof burden is still live.

Use the same rule to keep theorem honesty explicit.
An audited calibration dossier is theorem setup, not theorem delivery, when
the local record still lacks family-level formulas, extremal proofs, or a
transfer lemma back to the same burden currency.
In that situation, file the promoted object as a concrete next-test
assignment rather than as a completed theorem-facing result.
