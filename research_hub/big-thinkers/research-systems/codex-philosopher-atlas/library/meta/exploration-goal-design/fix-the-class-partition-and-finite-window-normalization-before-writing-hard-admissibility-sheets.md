# Fix The Class Partition And Finite-Window Normalization Before Writing Hard Admissibility Sheets

Before an exploration writes a hard admissibility sheet, first freeze the
class partition and the finite-window normalization.

Without those choices, a diagnostic vector turns into threshold bookkeeping
rather than an invariant gate, because its coordinates change meaning as the
partition or normalization drifts.
