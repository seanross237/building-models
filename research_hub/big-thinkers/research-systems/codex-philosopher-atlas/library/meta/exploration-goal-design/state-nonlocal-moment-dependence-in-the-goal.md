# State Nonlocal Moment Dependence In The Goal

If a branch depends on a nonlocal moment problem, say so directly in the
exploration goal.

Doing this early makes it easier to reject local claims that never touch the
live obstruction, especially geometry arguments that only control a local
subpiece while leaving the exteriorly generated tail unchanged.
