# When No Honest Survivor Remains, Use Enlargement To Kill The Tidiest Pseudo-Survivor And Keep The Verdict Budget-Limited

When a fixed budget already produces no honest survivor under the branch's
real closure rules, the enlargement pass still has one legitimate use:
test the tidiest artifact that would otherwise tempt the next step into
overclaim.

Run the smallest admissible enlargement that stays inside the same object
class and recompute closure from scratch. If the same spill or obstruction
comes back immediately, file that result as evidence that the tidy picture was
only a pseudo-survivor created by stopping closure too early.

That sharpened artifact check is still only a current-budget result. It does
not authorize a mission-level failure claim, and it does not self-authorize
an automatic search of the next budget rung.
