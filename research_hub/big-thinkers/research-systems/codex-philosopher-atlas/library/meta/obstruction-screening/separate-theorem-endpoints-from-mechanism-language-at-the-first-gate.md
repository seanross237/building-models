# Separate Theorem Endpoints From Mechanism Language At The First Gate

For endpoint-first obstruction branches, the first gate should sort candidate
phrases into two buckets before any further selection:

- actual theorem endpoints tied to one inherited contradiction ledger;
- and mechanism language that is only admissible if it changes the exact live
  bad term inside that ledger.

This prevents three common failure modes:

- treating bottleneck themes such as `full stretching control` as if they were
  already theorem statements;
- treating branded mechanism families such as
  `pressure-Hessian/strain defect control` as endpoints before they cash out;
- and letting tensor or representation language survive by renaming the
  obstruction instead of shrinking it.

When the local record already freezes one inherited ledger, the screen should
be explicit: the candidate is alive only if it acts on the named live term on
that same ledger. Otherwise it is endpoint drift, not a serious survivor.

Admission rules should be frozen against that exact endpoint object, not
against branch rhetoric. If the live theorem-level bad term is fixed, later
candidates do not survive by moving to a surrogate term, by rebranding a
package as leverage, or by invoking Tao sensitivity without changing the same
frozen estimate.

If the archive honestly supports only one theorem-facing floor, freeze the
endpoint window as one primary endpoint family plus at most one stricter
reserve, not as a menu of equal options. Continuation-style or norm-level
language should stay reserve-only until a candidate already behaves like
near-coercive control on the same inherited ledger.

Freeze solution class and localization regime together with that endpoint
window. Otherwise later candidate packets can smuggle architecture changes
into the branch under the label of mechanism gains.

On criterion-first geometry branches, make that sort concrete immediately:
freeze the endpoint to the most explicit theorem-facing criterion already named
on disk, not to a slogan such as `full stretching` unless the archive has
already promoted that slogan into a real theorem endpoint.

This also sharpens what it means to clear the gate. An endpoint-first branch
may survive narrowly once the endpoint object, gain currency, and admission
rules are frozen on the exact live term, but that does not yet admit any
tensor package. Prior-art and packaging screening still remain separate later
steps.
