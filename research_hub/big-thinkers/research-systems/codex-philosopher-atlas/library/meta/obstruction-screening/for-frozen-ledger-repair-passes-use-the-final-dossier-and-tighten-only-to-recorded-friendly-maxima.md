# For Frozen-Ledger Repair Passes, Use The Final Dossier And Tighten Only To Recorded Friendly Maxima

When a branch reaches a repair pass on one frozen exact ledger, treat the
final landed branch result as the controlling dossier.
Earlier intermediate reports may still matter for provenance or caution, but
they should not compete with the final landed verdict once the branch record
has already been closed.

Tighten each admissibility threshold only to the largest recorded friendly
value on that same ledger, and leave any already-saturated budget unchanged.
That removes bookkeeping slack without turning the repair pass into a fresh
family search or a post hoc threshold-invention exercise.

If the hostile dossier already earns a failure mode on the same frozen ledger,
keep that earned failure mode.
Do not fabricate a sharper hostile coefficient table just because the failure
is already known; repair from the recorded friendly margins and gaps instead.
