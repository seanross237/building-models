# Collapse Family Labels Before Equivalent-Packaging Screening On Endpoint-First Branches

When an endpoint-first branch has already frozen one theorem ledger, one live
bad term, and one gain currency, collapse family labels before package
selection.

Reusable rule:

- treat bottleneck slogans such as `full stretching control` as non-families
  unless they already carry an exact cash-out on the frozen ledger;
- treat mechanism subquestions such as harmonic-moment cancellation as part of
  the parent family unless they already define a distinct theorem-level gain;
- and treat alignment, eigenframe, restricted-Euler, or other observable
  wordings as overlap comparators or packaging aliases until they demonstrate
  estimate-level action on the same live bad term.

After that collapse, run the equivalent-packaging screen on the smallest honest
family menu. If no packaging returns to the same theorem-level inequality with
a smaller effective coefficient, stop the branch before tensor-package freeze.
