# The Exact Rewrite Branch Should Stop Before A Cosmetic Step 3

Status: `INFERRED`

Once the exact-rewrite branch is frozen to one bad term, one cutoff protocol,
and one gain currency, the honest Step-2 output is a continue-or-kill
decision.

For this branch, that decision is negative. Divergence/stress,
Lamb-vector/projected, and vorticity/Biot-Savart rewrites all fail to shrink
the fixed localized LEI cutoff-flux ledger, so opening a Step-3 estimate
ledger would only formalize candidates that have already failed the live
screen.

The branch should therefore stop at Step 2 and be invalidated on the current
architecture rather than prolonged by representation work with no surviving
estimate lever.
