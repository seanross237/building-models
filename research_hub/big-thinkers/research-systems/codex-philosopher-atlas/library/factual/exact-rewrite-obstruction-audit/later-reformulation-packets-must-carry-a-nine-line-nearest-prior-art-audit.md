# Later Reformulation Packets Must Carry A Nine-Line Nearest-Prior-Art Audit

Status: `INFERRED` Step-013 packet rule

For the Step-013 reformulation audit, prior-art comparison is not a loose
literature memo. Each later candidate packet must identify one nearest local
comparator, state the exact structure already shared with that comparator, and
show one same-currency cash-out on the frozen `I_flux[phi]` ledger or one
explicit transfer lemma that gets back there.

Minimum evidence to avoid immediate prior-art absorption:

- one exact nearest comparator on the repository record;
- one sentence naming the shared mechanism or identity already present there;
- one exact mathematical delta, not a branding claim;
- one frozen ledger entry improved after all debt is charged;
- one same-currency cash-out on the full `I_flux[phi]` bundle, or one explicit
  transfer lemma leading there.

The frozen audit format for later candidate packets is:

1. `candidate reformulation / unknown`
2. `nearest known comparator on the repository record`
3. `shared exact structure already present in that comparator`
4. `claimed mathematical delta`
5. `baseline burden entry being improved`
6. `debt ledger entries charged immediately`
7. `same-currency cash-out on full I_flux[phi] or explicit transfer lemma`
8. `anti-repackaging classification`
9. `prior-art absorption verdict`

If any required line is missing, the honest filing is immediate absorption or
an unresolved ambiguous packet rather than a promoted candidate.
