# Minimum Threshold Is Positive-Margin Net Shrinkage On The Same Full Pairing

Status: `INFERRED` with `PROPOSED` threshold formalization

Once the branch freezes the live endpoint ledger to the full far-field pairing
`I_p^far`, the minimum non-cosmetic Step-1 threshold is not generic harmonic
improvement, cleaner tensor packaging, or "better modulo constants."

Constants are already annihilated by the pairing, while affine-and-higher
harmonic moments remain live. Named cancellation therefore counts only if it
improves the same theorem-level inequality for the same full pairing.

The repository-supported threshold shape is a uniform positive-margin net
shrinkage of the live coefficient after later reconstruction and localization
debt are repaid, for example

`|I_p^far| <= (1 - delta) (||u||_{L^2}^2 / r_k^3) 2^{12k/5} U_k^{6/5}`

for some `delta > 0`, or an equivalent same-pairing estimate with a genuinely
smaller effective coefficient.

Anything weaker can leave the live obstruction coefficient unchanged and is
therefore only a surrogate or representation-level gain.
