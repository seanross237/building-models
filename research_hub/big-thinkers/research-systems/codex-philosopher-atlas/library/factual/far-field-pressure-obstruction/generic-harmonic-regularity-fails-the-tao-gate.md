# Generic Harmonic Regularity Fails The Tao Gate

Status: `VERIFIED`

For the harmonic-tail pressure branch, generic harmonic regularity does not
count as progress. Harnack, local `H^1`, Campanato control, derivative decay,
harmonic polynomial approximation, and similar elliptic upgrades remain inside
the harmonic-analysis structure that Tao's averaged Navier-Stokes model is
designed to preserve.

The branch standard is therefore stricter: harmonic regularity matters only if
it is tied to a specifically Navier-Stokes mechanism that shrinks the actual bad
coefficient in the full far-field pairing.
