# Projection Rigidity Is An Enforcement Mechanism, Not A Distinct Near-Closure Candidate

Status: `VERIFIED`

The current local record does not support `projection rigidity only` as a
fourth near-closure candidate.

Exact Leray / pressure projection is the enforcement mechanism that imposes
coefficient rigidity and spectator leakage across the promoted candidates, but
it does not by itself define a distinct near-closure object on this branch.

Keeping projection rigidity in a mechanism note prevents the Step-2 menu from
confusing object definition with the exact structural constraint that helps
enforce that object.
