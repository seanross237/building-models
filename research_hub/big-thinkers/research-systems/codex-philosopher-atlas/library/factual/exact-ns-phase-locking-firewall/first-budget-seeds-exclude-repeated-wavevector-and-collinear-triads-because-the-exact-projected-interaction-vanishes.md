# First-Budget Seeds Exclude Repeated-Wavevector And Collinear Triads Because The Exact Projected Interaction Vanishes

Status: `INFERRED` seed-definition screen from the frozen exact interaction
sheet

At the one-triad Step-2 budget, not every formal relation
`k = p + q`
is an honest seed.
If `p = q`, the branch's exact projected interaction scalar collapses because
divergence-free modes satisfy `p · u^(p) = 0`.
If `p` and `q` are collinear, the same orthogonality forces
`q · u^(p) = 0`.

So repeated-wavevector and collinear relations do not define active exact
triads with nonzero projected coefficient on the frozen support sheet.
They should be excluded from the first-budget seed catalog itself, not logged
later as failed closure survivors or dead-end exact seeds.
