# Intrinsic Phase-Coherence Objects Must Pass A Five-Layer Helical Admissibility Screen

Status: `VERIFIED` mission constraint with `INFERRED` branch-level quotient
rule

For the exact-NS phase-locking branch, an admissible phase/coherence object
must be built from exact Fourier/helical interaction data and tested modulo
five layers:

- triad-orbit relabeling, including cyclic permutation of `(k,p,q)` and
  equivalent helical-orbit representatives together with the matching exact
  coefficient change;
- real-valued conjugate completion;
- helical-basis phase and sign conventions;
- gauge or anchor conventions such as phase-anchor or translation-gauge
  choices;
- amplitude normalization.

An object is `intrinsic` only if it is invariant under all five layers and its
raw definition does not name desired channels or other target annotations.
It is `canonically representable but non-intrinsic` if it becomes honest only
after fixing one aggregation or normalization sheet.
It is `inadmissible` if the raw formula already imports desired channels,
packet roles, frozen itinerary content, or another hoped-for transfer label.
