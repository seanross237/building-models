# Exploration 001 Goal — Prior-Art Benchmark for the Geometry Route

## Objective

Benchmark the geometry-route candidate families against the prior-art geometry frameworks explicitly named in the local repository context, and state where this branch would merely restate that literature rather than add a mission-live ingredient.

Use the local corpus to calibrate three candidate families:

- vorticity-direction coherence / direction-regularity routes
- geometric depletion / local Beltrami or alignment routes
- tube coherence or persistence routes

The benchmark must answer, for each family:

- what framework the local corpus says already exists
- what estimate-level or claim-level boundary that framework earns
- what overlap would make a new claim only a relabeling
- what gap remains before the chain could honestly call the ingredient live

## Success Criteria

- Produce a source-based benchmark memo, not a generic survey.
- Separate `[VERIFIED]` repository-supported claims from `[INFERRED]` calibration judgments.
- Name exact source files for every major benchmark claim.
- State at least one explicit "forbid novelty claim" rule for each candidate family.
- End with a bounded verdict on whether the branch still has any ingredients worth carrying into Step 2.

## Failure Criteria

- Treats planning rhetoric as if it were already a theorem claim.
- Calls a route novel without stating what prior-art framework it overlaps.
- Uses descriptive vortex language without tying it to a concrete claim boundary.
- Ignores the pressure-side obstruction memo as the main negative reference point for static singular-integral geometry.
