# Exploration 001 Goal

## Objective

Freeze exactly one theorem-facing endpoint family, one gain currency, and one
candidate proposition / contradiction line / insertion claim for step-015
before any package is chosen.

## Success Criteria

- choose one endpoint family only:
  Constantin-Fefferman-type criterion by default, or one named
  full-stretching contradiction target only if the local record supports it
  more concretely;
- choose one gain currency only and one minimum non-cosmetic gain in that same
  currency;
- write one candidate proposition, contradiction line, or insertion claim in
  endpoint-facing language before package choice; or
- show from exact local sources that no such claim is honestly writable yet.

## Failure Criteria

- leaves multiple endpoint families live;
- mixes criterion-input and full-stretching currencies without choosing one;
- writes only a geometry theme rather than a theorem-facing claim; or
- depends on package selection to state the claim at all.
