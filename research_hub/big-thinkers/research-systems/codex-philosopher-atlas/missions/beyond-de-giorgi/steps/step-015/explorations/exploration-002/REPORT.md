## Findings

- [VERIFIED] The only locally frozen theorem floor remains the inherited
  suitable-weak / Leray-Hopf plus LEI framework.
- [VERIFIED] The Step-3 bounded observable table already writes the live hybrid
  on theorem-floor objects inside the fixed Eulerian package:
  the intense set `E_r(t) = {x in B_r(x_*) : |omega(x,t)| >= r^-2}` on
  `B_r(x_*) x [t_* - r^2, t_*]`, together with direction data on that set.
- [VERIFIED] No named proxy class is frozen anywhere in the required local
  record for this geometry route.
- [INFERRED] The honest admissibility freeze is therefore:
  `direct suitable-weak / LEI admissible`, not proxy-based.
- [INFERRED] The important caveat is different:
  the route is transport-dependent rather than genuinely geometry-native on the
  theorem floor. Its only live content is the hoped-for persistence of
  direction coherence along a coherent tube family across the diffusion-scale
  window.

### 2. Exact bridge burden back to the theorem floor

- [INFERRED] Because no proxy class is honestly frozen here, there is no adopted
  proxy-to-floor bridge ledger.
- [INFERRED] The real burden sits inside the theorem floor itself and has three
  inseparable parts:
  1. match coherent intense components across the full `r^2` window inside the
     neutral Eulerian package without tube-adapted relocalization, threshold
     retuning, or hidden axis fitting;
  2. propagate direction coherence on that moving intense set without importing
     the stronger `nabla xi`-type derivative control that the local record
     treats as a return to known direction-regularity burden; and
  3. turn that persistence information into a smaller criterion-scale input or
     a smaller full-stretching contribution on the inherited theorem floor.
- [VERIFIED] The local record already identifies exactly these burdens as the
  missing mechanism behind the hybrid's failure:
  Eulerian component matching,
  diffusion-scale persistence,

  and `nabla xi`-type debt.
- [INFERRED] If one tried to reinterpret those burdens as a de facto proxy
  bridge, it would already look `theorem-scale`.
