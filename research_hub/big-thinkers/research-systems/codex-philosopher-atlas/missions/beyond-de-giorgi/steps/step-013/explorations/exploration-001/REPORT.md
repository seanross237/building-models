## Findings

### 1. What theorem-facing floor is actually frozen

- [VERIFIED] The only locally frozen theorem-facing floor remains suitable weak
  solutions in the Leray-Hopf energy class with LEI, together with the
  CKN/Lin epsilon-regularity contradiction package.
- [VERIFIED] No critical-space or adapted-package replacement endpoint is
  frozen on disk for this branch.
- [VERIFIED] Step-013 therefore has to freeze its endpoint window inside that
  inherited LEI / CKN floor rather than reopening framework choice.

### 2. Primary endpoint family

- [VERIFIED] Run-009 requires one inherited theorem-facing endpoint on the
  suitable-weak / LEI floor before any candidate reformulation work begins.
- [VERIFIED] The run-009 selected-chain materials say that this endpoint is
  normally `local concentration exclusion inside the CKN/Lin contradiction package`.
- [INFERRED] The correct primary endpoint family for this chain is therefore:
  `local concentration exclusion / endpoint-regularity exclusion on the inherited CKN/Lin LEI ledger`.
- [INFERRED] Why this is the right first window:
  it is the only endpoint family on disk that is both theorem-facing and
  already attached to the inherited contradiction line; everything else is
  mechanism language or a stronger reserve.

### 3. Backup endpoint family

- [VERIFIED] The inherited record repeatedly treats continuation-style or
  norm-level control as stricter than concentration exclusion, not as the
  default first endpoint.
- [VERIFIED] The step-012 endpoint-admission record freezes the rule:
  `continuation only after near-coercive norm-level behavior`.
- [INFERRED] The correct backup endpoint family is therefore:
  `continuation / norm-level control only if a candidate already behaves like near-coercive control`.
- [INFERRED] Why this is the right reserve:
  it keeps exactly one harder backup alive without widening the menu into a
  soft portfolio, and it preserves the rule that the branch must earn a move
  away from concentration exclusion rather than assume it.

### 4. Solution class freeze

- [VERIFIED] The correct solution-class floor is:
  suitable weak solutions in the Leray-Hopf energy class with LEI.
- [VERIFIED] The non-negotiable inherited data are:
  divergence-free velocity,
  `u in L^infty_t L^2_x cap L^2_t H^1_x`,
  pressure in the suitable-weak sense,
  and the local energy inequality.
- [VERIFIED] A candidate that needs a stronger host class than this has left
  the branch's admitted theorem floor and must pay an admissibility debt.

### 5. Localization regime freeze

- [VERIFIED] The fixed localization regime is one CKN-style parabolic cylinder
  with one shared smooth cutoff.
- [VERIFIED] The protocol is:
  `Q_r(x_*, t_*) = B_r(x_*) x (t_* - r^2, t_*)`,
  one `phi in C_c^\infty(Q_r)` with the standard CKN derivative bounds,
  and one shared localized LEI package for every later candidate.
- [VERIFIED] Every later reformulation must pay projection, CZ, pressure
  recovery, Biot-Savart reinsertion, and cutoff-commutator costs inside this
  same localization regime.

### 6. Endpoint-pair memo

- [VERIFIED] Actual theorem endpoints and mechanism language must be separated
  at this gate.
- [INFERRED] Primary endpoint:
  `local concentration exclusion / endpoint-regularity exclusion on the inherited LEI / CKN ledger`.
- [INFERRED] Backup endpoint:
  `continuation / near-coercive norm-level control`, admissible only as a reserve

- [VERIFIED] Decorative endpoint claims to reject at Step 1:

  candidate framed as a mechanism without an endpoint cash-out on the inherited
  contradiction line.
