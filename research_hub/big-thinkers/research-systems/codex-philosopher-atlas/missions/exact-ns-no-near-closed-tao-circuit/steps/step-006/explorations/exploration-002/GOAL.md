# Exploration 002 Goal - Attach The First Exact Next Tests

## Objective

Using the repaired Step-5 shortlist and the frozen packet ledger, attach one
first exact theorem question or one first counterexample-search question to
each promoted Step-6 object.

The exploration must name the smallest meaningful carried-forward test object
or parameterized family, the exact invariant observable or admissibility sheet
for the next phase, and the exact data still missing before a theorem or
counterexample result would be honest.

## Required Output

Produce a report that does all of the following:

1. For repaired `Template-Defect Near-Closure`, states one first exact theorem
   question or one first counterexample-search question on the frozen packet
   ledger.
2. For repaired `Windowed Spectator-Leakage Budget`, states one first exact
   theorem question or one first counterexample-search question on the same
   frozen ledger.
3. For each assigned question, names:
   - the smallest meaningful carried-forward test object or parameterized
     family;
   - the invariant observable or admissibility sheet that the next phase must
     use; and
   - the exact missing data still needed before the next phase could claim a
     theorem or counterexample result.
4. States the strongest claim each promoted object is allowed to support now
   and what the local record still does not justify.
5. Ends with one Step-6 handoff memo:
   the exact next-test assignments mission control should treat as the branch
   handoff if the survivor set remains live.

## Success Criteria

- The questions stay inside the audited packet class and fixed window.
- The carried-forward object or family is no larger than needed for the first
  honest next test.
- Missing data are specific and checkable rather than generic requests for
  "more theory" or "more computation."

## Failure Criteria

- The exploration fails if it assigns a theorem question that already
  overclaims beyond the audited witnesses or packet class.
- It fails if it assigns a counterexample search without naming the invariant
  sheet or admissibility rules the search must respect.
- It fails if it confuses a package-level insertion line with an earned
  theorem result.
