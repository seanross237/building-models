F_DT trigger upper bound on the frozen normalized window I = [0, 1]
Assumption: |C'(t)| <= K_seed * delta + K_amp * |C(t)| and |C(0)| = eta.

Then Gronwall gives
  |C(t)| <= exp(K_amp) * (eta + K_seed * delta)  for 0 <= t <= 1.

On the explicit subregime eta <= delta^2 and delta <= 1, this simplifies to
  |C(t)| <= exp(K_amp) * (delta^2 + K_seed * delta)
         <= exp(K_amp) * (K_seed + 1) * delta.

Therefore the sufficient condition
  delta <= 1 / (4 * exp(K_amp) * (K_seed + 1))
implies
  |C(t)| <= 1/4 = theta_C on I,
so no trigger event occurs on the frozen window:
  t_trig > 1.
