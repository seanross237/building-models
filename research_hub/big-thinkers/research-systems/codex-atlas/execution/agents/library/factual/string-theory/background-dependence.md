# Background Dependence in String Theory

A major criticism of string theory as a quantum gravity theory is that it is not manifestly background-independent. General relativity treats spacetime geometry as fully dynamical, but perturbative string theory is formulated on a fixed background spacetime, with other geometries described as perturbations around this fixed reference.

## The Criticism

Lee Smolin (Perimeter Institute) has been the most prominent critic on this point, arguing in "The Case for Background Independence" (arXiv:hep-th/0507235) that background independence is essential for any theory of quantum gravity. His core argument:

1. General relativity's central insight is that spacetime geometry is dynamical, not a fixed stage.
2. A quantum theory of gravity must preserve this insight -- the theory's equations should not refer to any preferred background geometry.
3. String theory's formulation on fixed backgrounds fails to incorporate this lesson.
4. The landscape problem (inability to select a unique vacuum) may be a direct consequence of background dependence -- without background independence, there is no dynamical mechanism to select among vacua.

## The Defense

String theorist Joseph Polchinski countered that Smolin mistakes a feature of the mathematical formalism for a physical limitation. The perturbative formulation uses a fixed background as a calculational tool, but the physics described is background-independent in the same way that choosing coordinates in GR doesn't make GR coordinate-dependent.

## Comparison with Loop Quantum Gravity

Loop quantum gravity is manifestly background-independent and non-perturbative from the start -- spacetime geometry is built from quantum states of geometry (spin networks) without reference to any background metric. This represents a fundamental conceptual difference in approach.

## M-Theory and Background Independence

The expectation among string theorists is that M-theory (the conjectured non-perturbative completion unifying all five superstring theories in 11 dimensions) would be background-independent, with the various background-dependent string theories emerging as specific limits or solutions. However, M-theory lacks a complete formulation, so this remains a conjecture.
