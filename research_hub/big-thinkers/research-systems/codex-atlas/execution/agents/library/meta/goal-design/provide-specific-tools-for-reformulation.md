## Lesson

For reformulation or extension explorations — where the goal is to re-derive an existing result
using a different framework — specify the exact mathematical tools to use in the goal. Don't ask
the explorer to survey possible approaches and select one. Providing the specific tool set (e.g.,
"use the Ryu-Takayanagi formula and holographic quantum error correction") lets the explorer
dive directly into the mapping rather than spending turns on landscape surveys.

## Evidence

- **classicality-budget strategy-001 exploration 007** — Providing the specific tools (RT formula,
  HQEC, HaPPY code, entanglement wedge reconstruction) allowed the explorer to produce a
  4-part structure (mapping → derivation → assessment → literature) in a single exploration,
  covering both a formal derivation and a detailed literature search for prior connections. Without
  pre-specifying the tools, this would have required at least one survey exploration first.

## When to apply

When a strategizer has already identified WHICH framework to attempt a reformulation in (e.g.,
"reformulate in holographic language," "re-derive using optimal transport," "extend to curved
spacetime using the Bogoliubov transformation"). If the strategizer doesn't yet know which
framework to try, then a survey exploration is appropriate first (see `divergent-survey-pattern.md`).

## Contrast: When Not to Apply

When the exploration goal is to FIND which approach might work (the search IS the task), pre-specifying
tools would artificially narrow the search. In that case, the divergent survey pattern is more appropriate.
