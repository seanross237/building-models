## Lesson

When designing a goal to assess the physical content or value of a reformulation, include a concrete successful predecessor reformulation as a structured comparison framework. Don't just ask "does this reformulation add value?" -- ask "does this reformulation add value comparable to [amplituhedron / AdS/CFT / etc.]?" The comparison produces a multi-dimensional evaluation table that is far clearer than a standalone philosophical assessment.

## Evidence

- **barandes-stochastic strategy-001 exploration-004** -- The goal provided the amplituhedron as a concrete precedent for "reformulation that adds value." The explorer used it as a structured framework: UV finiteness -> Tsirelson bound, EFT-hedron -> no analog, hidden zeros -> possible but undeveloped, simpler computations -> no. The comparison table in Section 5 was the clearest, most actionable part of the report. Without the precedent, the assessment would have been a vague philosophical discussion.

## Variant: "One Paragraph Answer to the Key Question"

Require the explorer to produce a one-paragraph answer to the central evaluation question (e.g., "What does this framework buy a working physicist?"). This forces concrete, honest synthesis. The resulting paragraph from E004 was publishable-quality and is now the definitive summary of the ISP framework's value.

**Design template:** Add to GOAL.md: "Produce a single paragraph, addressed to a skeptical working physicist, answering: 'What does [framework] buy me?' Be concrete and honest."

## Variant: Rank Papers by Importance in Cluster Assessments

When a cluster of papers exists from the same program, ask the explorer to RANK them by physical significance rather than just listing them chronologically. In E004, the CHSH/Tsirelson paper (Dec 2025) emerged as the strongest finding -- this was unexpected given prior explorations focused on core framework papers. Ranking forces the explorer to make comparative judgments that expose the most important work.

**Design template:** "Rank the papers by physical significance (strongest first). For each, state: does it add physical content beyond the prior literature? If so, what specifically?"

## When to Apply

- Any goal assessing whether a reformulation adds genuine value beyond the original formalism
- Synthesis explorations evaluating a research program's contributions
- Paper cluster assessments where multiple related papers need comparative evaluation

## When NOT to Apply

- When no suitable predecessor reformulation exists in the domain
- When the goal is computation or derivation, not evaluation
