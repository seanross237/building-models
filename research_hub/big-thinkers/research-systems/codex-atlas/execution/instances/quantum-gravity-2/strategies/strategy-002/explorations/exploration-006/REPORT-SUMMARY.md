# Exploration 006 Summary: Deep Build — Causal Fakeon Theory (CFT)

## Goal
Build Causal Fakeon Theory (CFT) — a focused theory connecting spacetime's causal partial order to the fakeon prescription in quadratic gravity. State axioms, derive consequences, address the loop-level problem, compare to QG+F, identify novel predictions.

## What Was Done
Constructed CFT from four axioms and one conjecture:
- **Axiom 1 (Causal Foundation):** Spacetime is fundamentally a causal set; causal order determines conformal geometry (Malament) and volume measure determines the conformal factor (Sorkin).
- **Axiom 2 (Spectral Causal Compatibility):** The SJ vacuum construction determines propagators from causal structure. Modes with negative spectral weight (ghosts) break the SJ construction — the causal order cannot support them as physical particles.
- **Axiom 3 (Wheeler Exclusion):** Excluded modes propagate via G_W = (G_R + G_A)/2, which equals the fakeon propagator at tree level. Proven independently by Belenchia et al. (2015) for causal set unstable modes.
- **Axiom 4 (Perturbative Self-Consistency):** The fakeon's purely virtual nature ensures it doesn't invalidate the background causal order used to define it.
- **Conjecture (Loop Extension):** The tree-level derivation extends to loop level, reproducing Anselmi's threshold-dropping procedure. UNPROVEN — this is CFT's make-or-break challenge.

Derived five physical consequences (QG+F recovery, GR limit, microcausality as discreteness, d_s=2 unification, analyticity explanation). Built comparison table. Identified three novel predictions (dimensional crossover profile, stochastic microcausality violation, quadratic gravity from causal sets).

## Key Takeaway

**CFT provides genuine explanatory value for QG+F's #1 debt (fakeon interpretation) by deriving the fakeon from causal order rather than postulating it.** The tree-level connection G_fakeon = G_Wheeler is mathematically exact, and the causal set origin is independently proven (Belenchia et al. 2015). CFT also explains microcausality violation (as discreteness), analyticity sacrifice (as causal enforcement), and d_s=2 universality (as causal set dynamics).

**The critical weakness is the loop-level conjecture.** Without it, CFT is a tree-level motivation, not a complete derivation. The conjecture is well-posed and generates five specific open problems for future work. CFT is best understood as a research program connecting causal sets to QG+F, not as a competitor to QG+F.

## Leads Worth Pursuing
1. **Loop Extension:** The interacting SJ construction on causal sets (Dowker et al. 2024, arXiv:2402.08555) is the most promising route.
2. **Spin-2 SJ Construction:** Extending SJ from scalars to tensor fields — needed for CFT to apply to the actual ghost.
3. **Quadratic Gravity from Causal Sets:** Finding causal set actions that produce R² + C² terms — a falsifiable structural prediction.
4. **Crossover Profile Computation:** Computing d_s(E) from both QG+F and causal set d'Alembertian to determine if they differ — the closest thing to a distinguishing quantitative prediction.
