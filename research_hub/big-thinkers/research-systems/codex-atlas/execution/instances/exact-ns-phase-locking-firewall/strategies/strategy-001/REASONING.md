## Iteration 0 decision

This mission is unstarted. The strategy is designed to front-load the intrinsic
definition gate and the smallest-support audit because the predecessor exact-NS
missions died when the object was either non-canonical or too detached from the
exact dynamics it was supposed to explain.

So the strategizer should not begin with a broad survey of phase ideas. It
should freeze one preferred phase object, freeze one preferred delayed-transfer
event, and then test the cheapest honest exact setting before spending budget on
any longer forcing or counterexample campaign.

1. test whether one preferred intrinsic exact triad-phase/coherence observable

## Iteration 1 result

Exploration 001 did **not** end in the expected immediate definition failure.
Instead it found a narrow but usable exact phase object:

```text
C_ℓ(t) = (∑_{τ∈𝒯_ℓ} T_{τ,ℓ}(t)) / (∑_{τ∈𝒯_ℓ} |T_{τ,ℓ}(t)|),
D_ℓ(t) = 1 - C_ℓ(t),
```

where `T_{τ,ℓ} = W_{τ,ℓ} cos Θ_{τ,ℓ}` are the exact signed helical triad
contributions to one smooth scale-interface transfer ledger. A backup raw phase
concentration observable `A_ℓ` also survives but is weaker.

The important scientific shift is that the preferred object no longer depends
on Tao-role hypergraphs, packet representatives, or desired-triad witness sets.
The triad phase variable is intrinsic exact-NS data, and the population is just
the sign-closed exact transfer ledger across `ℓ -> ℓ/ρ`.

The pass is still narrow:

- the old packet-sign family remains dead,
- localized physical-space `E_flux` is only an imperfect companion because the
  spatial cutoff obscures the exact triad ledger,
- the clean delayed-transfer companion is the global spectral event
  `E_ℓ^transfer`,
- and the main risk has shifted from definition-level failure to
  **model-level failure in the smallest honest setting**.

Operational note: the launched explorer stalled after writing only the report
skeleton. Manual recovery was straightforward because the decisive ingredients

## Iteration 2 result

Exploration 002 closed the strategy.

The frozen pair `(C_ℓ, E_ℓ^transfer)` fails at the model level in the smallest
honest exact setting:

1. on one active exact transfer term, `C_ℓ = ±1` identically, so the observable
   is too tautological to express a real phase-locking burden;
2. on the first nontrivial exact-helical cluster with two independent transfer
   inputs feeding the same receiver mode, exact phase tuning keeps net
   receiver-band gain positive while driving `C_ℓ` arbitrarily close to `0`.

That means the observable is defeated in both minimal regimes that matter:

- too trivial when the support is smallest,
- too weak as soon as the support is large enough to make the question
  nontrivial.

So the correct terminal mission outcome is:

```text
model-level failure
```

No Phase 3 forcing statement or adversarial screen beyond the minimal audit is
scientifically justified. The strategy should end here.
