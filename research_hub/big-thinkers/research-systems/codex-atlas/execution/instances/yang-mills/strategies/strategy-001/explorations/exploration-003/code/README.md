# SU(2) Lattice Gauge Theory — Code

## Files

- `su2_lattice.py` — Core implementation: SU(2) quaternion algebra, lattice data structure, measurements (pure Python with numpy)
- `su2_simulation.py` — Numba-optimized Monte Carlo simulation with heat bath and full measurement suite. **Run this first.**
- `analysis.py` — Extract physics: static potential, string tension, scaling. Run after simulation.
- `glueball_mass.py` — Glueball mass extraction via multiple methods. Run after simulation.
- `verification.py` — Cross-check against literature. Run after simulation.
- `results.json` — Raw results from simulation (generated by su2_simulation.py)

## Dependencies

- Python 3.8+
- numpy
- numba
- scipy (for verification only)

## Run Order

```bash
python su2_simulation.py    # ~5 minutes, produces results.json
python analysis.py          # Physics extraction
python glueball_mass.py     # Glueball mass analysis
python verification.py      # Cross-checks
```

## Key Parameters

Edit the `configs` list in `su2_simulation.py` to change lattice sizes, beta values, and statistics.
