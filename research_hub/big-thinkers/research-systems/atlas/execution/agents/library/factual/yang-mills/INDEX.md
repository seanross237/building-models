## Core Results

- **balaban-uv-stability.md** — Balaban's 14-paper RG program (1983-1989): proves UV stability of 4D lattice pure Yang-Mills in finite volume via four-phase construction (foundational tools → 3D proof → small-field 4D RG → large-field analysis). Precise theorem: partition function bounded uniformly as lattice spacing → 0. Also completes 3D UV stability (superrenormalizable case).

- **gap-structure-overview.md** — Two-tier gap structure toward the Millennium Problem: Tier 1 (potentially tractable) = observable control on T⁴ + uniqueness; Tier 2 (fundamental obstruction) = mass gap + infinite volume. Classification table (8 results: numerical vs. rigorous status), 12-step reconstruction chain, 7-step logical dependency chain, OS axioms requirements. Four specific gaps with mathematical formulations.

- **lattice-numerical-evidence.md** — 50 years of numerical lattice results: confinement with √σ ≈ 420 MeV, mass gap m(0++) ≈ 1.7 GeV, full glueball spectrum, asymptotic scaling (β = 5.7–7.5), universality across actions, deconfinement at T_c ≈ 270 MeV (SU(3)), large-N scaling for N = 2–12. Continuum limit procedure (scale setting, extrapolation). **Atlas SU(2) verification** (exploration-003: first-principles computation on 4⁴-8⁴ lattices reproduces known results within 1-2σ; practical lattice size requirements table; glueball extraction needs ≥16⁴). ALL numerical only — defines what rigorous proof must reproduce.

- **proof-strategies-comparison.md** — Comparative assessment of 5 proof strategies: (1) complete Balaban (most direct, 10–20 yr, no mass gap), (2) stochastic quantization/Hairer (d=4 barrier fundamental), (3) probabilistic/Chatterjee (**most active**, 5–15 yr), (4) large-N/'t Hooft (elegant but finite-N extraction unclear), (5) hybrid lattice+functional analysis (speculative). No strategy has clear path to mass gap for continuous groups. **Timeline assessment: 20-50+ year problem.** Breakthrough tier classification (Tier 1-3). Wild cards (ML, SUSY bootstrap, conformal bootstrap, tensor networks).
