# Correctness Suite

This folder is for correctness-oriented evaluation.

It should stay separate from grading and later protect dry-run correctness and behavior-preserving changes.

Current v1 utility:

- `validate_run_v1.py`

Run it like this:

```bash
python3 data-system/correctness-suite/validate_run_v1.py data-system/run-history/run_2026_04_04_smoke
```
