# Gardener

Future expansion.

Gardener is a model-guided learning component inside Bonsai that reviews runs and proposes better variable choices.

It should be able to:

- inspect runs
- notice failure patterns or missed opportunities
- propose new values for existing variables
- propose new candidate variables when needed
- suggest experiments to try those changes

Gardener is meant to be a shorter-term learning layer before more mature ML-based variable selection.
