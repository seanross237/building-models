Open-Eywa is a system designed for executing and managing computational tasks. In Open-Eywa, a node serves as the durable unit of work. This means each node encapsulates a specific task or a part of a larger mission, allowing for persistence, resumption, and independent execution. For a human operator inspecting a mission, the `input/goal.md` file is the primary document to review first, as it outlines the core objective of the mission.
