# Round 1 Design Questions

These are the highest-leverage questions to settle before writing the detailed target docs:

1. What is the real optimization surface?
2. What must every node always expose?
3. What actions may a node take?
4. Where should specialization live?
5. What data should every run emit?
6. How should tasks be represented before ML?

This round is intentionally not trying to settle Potter internals, storage engine choices, or later ML details.
