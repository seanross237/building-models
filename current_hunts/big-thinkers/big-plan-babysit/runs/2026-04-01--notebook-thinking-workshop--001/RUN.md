# Run: Notebook Thinking Workshop

## Goal

Create a concise internal brief for a fictional workshop called `Notebook Thinking`, including:

- a target audience
- a clear workshop promise
- a three-session outline
- a one-paragraph pitch

## Done When

- there is a final brief in the run folder
- the brief includes the required four sections
- the workshop concept is coherent and internally consistent

## Policies

- `human_review_policy: none`
- `stop_policy: major_decisions`
- `replan_policy: local_first`
- `planning_ensemble.enabled: true`
- `planning_ensemble.run_on: [initial_plan]`

## Constraints

- keep the toy run small and fast
- use subagents for planning and phase execution
- leave a resumable artifact trail in the run folder
