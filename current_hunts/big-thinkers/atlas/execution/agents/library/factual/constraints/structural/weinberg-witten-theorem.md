# Weinberg-Witten Theorem Compliance (Constraint A7)

**Statement:** Massless particles with spin j > 1 cannot carry a Lorentz-covariant stress-energy tensor T^μν. This constrains theories where the graviton is a composite or emergent particle — the graviton cannot be a bound state in a Lorentz-covariant quantum field theory with a conserved T^μν.

**Mathematical form:**
If |p, σ⟩ is a one-particle massless state with spin j > 1 and a conserved, Lorentz-covariant T^μν exists, then ⟨p', σ'|T^μν|p, σ⟩ = 0, meaning the particle doesn't gravitate — contradiction.

**Restrictiveness: MEDIUM-HIGH**

This constrains emergent gravity programs significantly. Known escape routes:

1. **Holographic/AdS-CFT:** The graviton lives in a different (bulk) spacetime from the boundary theory — the theorem's assumptions about the graviton being in the same spacetime as T^μν don't apply
2. **Broken Lorentz invariance:** If the UV theory breaks Lorentz symmetry, the theorem's Lorentz-covariance assumptions fail
3. **Gauge invariance of GR:** In GR itself, the stress-energy of the gravitational field is not Lorentz-covariant (it transforms as a pseudo-tensor), so the theorem does not apply to GR

## Significance for Theory Construction

Any emergent gravity program must identify which escape route it uses. The holographic escape (graviton in bulk, emergent from boundary degrees of freedom in a lower-dimensional space) is the most commonly employed.

## Dependency

Genuinely independent — follows from Lorentz invariance and conservation laws, not from other QG constraints.
