# Causal Set Theory and Lorentz Invariance

A central achievement of causal set theory is that its fundamental discreteness does not violate local Lorentz invariance in the continuum approximation. This is remarkable because most approaches to discrete spacetime (lattices, spin foams) explicitly break Lorentz symmetry by introducing a preferred frame or direction.

## The Problem with Naive Discreteness

A regular lattice of spacetime points selects a preferred rest frame -- the frame in which the lattice is regular. This breaks Lorentz invariance at the discreteness scale, potentially producing observable violations at accessible energies (e.g., modified dispersion relations for photons). No such violations have been observed.

## The Causal Set Solution: Poisson Sprinkling

CST avoids this by using a random (Poisson) distribution of points rather than a regular lattice. The sprinkling process works as follows:

1. Points are selected from a Lorentzian manifold via a Poisson process with density rho (approximately the Planck density).
2. The causal order is inherited from the manifold's light-cone structure.

Both steps are manifestly Lorentz invariant: the Poisson process depends only on the spacetime volume element (a Lorentz scalar), and the causal order depends only on the causal structure (also Lorentz invariant).

## The Bombelli-Henson-Sorkin Theorem

Bombelli, Henson, and Sorkin (2009) proved that not only is the sprinkling process Lorentz invariant as a statistical ensemble, but almost all individual causal sets produced by sprinkling into Minkowski spacetime do not select a preferred Lorentz frame. This is a stronger result than mere statistical invariance: it shows that individual realizations of the discrete structure respect the symmetry.

The key insight is that randomness is not a defect but an essential feature: it is precisely the stochastic nature of the sprinkling that prevents the discrete structure from "seeing" any preferred direction in spacetime.

## Characteristic Non-Locality

The price of maintaining Lorentz invariance with discreteness is a characteristic non-locality that distinguishes CST from other quantum gravity approaches. This non-locality arises because there is no invariant notion of "nearest neighbor" on a Lorentzian causal set -- in a sprinkled Minkowski causal set, every element has infinitely many "nearest neighbors" (links) due to the infinite volume of the light cone boundary.
