# CDT Phase Diagram and Gravitational Phase Transitions

## CDT Phase Structure

Causal Dynamical Triangulations exhibits a rich phase diagram analogous to lattice QCD:

| Phase | Properties | QCD Analogue? |
|-------|-----------|---------------|
| **Phase A (branched polymer)** | Spacetime degenerates, no macroscopic extension | Deconfined/high-T phase? |
| **Phase B (crumpled)** | Spacetime collapses, very high Hausdorff dimension | Strong coupling phase? |
| **Phase C (de Sitter)** | Extended 4D spacetime, d_s runs 4 -> ~2 | Confined/hadronic phase |
| **Phase C_b (bifurcation)** | Between B and C, fourth phase | No clear analogue |

## Phase Transitions

- **A-C transition:** First order (discontinuous). Recent 2024 work shows UV critical behavior near this transition line -- coupling runs correctly when approaching it.
- **B-C transition:** Second order (continuous). This is the candidate for defining a UV continuum limit -- the analogue of the UV fixed point in asymptotic safety.

## QCD Analogy for Phases

In QCD, the deconfinement phase transition at T ~ 150 MeV separates confined (hadrons) from deconfined (quark-gluon plasma).

In CDT, the Phase C -> Phase A transition could be the gravitational analogue:
- Phase C: "confined" phase where 4D spacetime emerges (GR limit)
- Phase A: "deconfined" phase where spacetime structure dissolves

However, the physical interpretation is unclear. The "deconfined" phase would require trans-Planckian energies and may not correspond to any physically realizable state.

## Spectral Dimension as Phase Transition

The d_s = 4 -> d_s ~ 2 flow can be interpreted as a dimensional phase transition:
- **IR phase (d_s = 4):** Classical spacetime, GR applies
- **UV phase (d_s ~ 2):** Quantum spacetime, scale-invariant at fixed point
- **Crossover scale:** ~ Planck scale

CDT evidence suggests the B-C transition is second order, meaning the d_s crossover is a **genuine phase transition** with correlation length diverging at the critical point. This would mean the Planck scale marks an actual critical point, not just a crossover.

## CDT-FRG Comparison (2024)

Paper in PRD 110, 126006 (2024): "Is lattice quantum gravity asymptotically safe? Making contact between CDT and the FRG" -- first attempt to quantitatively compare CDT lattice results with functional RG results from AS. Result: **"the two results are not yet in good agreement"** -- an important tension between the lattice and continuum approaches.

## Recent Developments

1. **CDT UV fixed point (2024):** Ambjorn et al. found coupling behavior near the A-C_dS transition line consistent with UV critical behavior, strengthening the case for CDT defining a UV continuum limit consistent with AS.
2. **Inflationary phase transitions (2024-2025):** Multiple papers study phase transitions during inflation producing gravitational wave signatures detectable by future space-based detectors (LISA).
