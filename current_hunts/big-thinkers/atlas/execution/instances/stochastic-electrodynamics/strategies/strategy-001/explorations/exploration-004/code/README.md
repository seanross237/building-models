# Code: Exploration 004 — ALD Simulation

## Files

- `ald_simulate.py` — Main simulation. Implements the Landau-Lifshitz order-reduced Abraham-Lorentz equation with position-dependent damping Γ_eff = τ(ω₀²+12βx²) and noise derivative τF'_zpf(t).
- `analyze_ald.py` — Analysis script. Produces 3-way comparison table and order-of-failure power-law fit.
- `ald_results.json` — Results for all β values (generated by ald_simulate.py)
- `ald_samples_beta*.npy` — Position samples for each β (generated by ald_simulate.py)

## Running

```bash
# Run single β value
python ald_simulate.py 0.0
python ald_simulate.py 0.01
python ald_simulate.py 0.1
python ald_simulate.py 1.0

# Run analysis (requires ald_results.json)
python analyze_ald.py
```

## Parameters

- τ = 0.01, ω₀ = 1, ω_max = 10
- dt = 0.05, T = 20000 (400,000 steps per trajectory)
- N_ensemble = 200, batch_size = 50
- 50 samples per trajectory → 10,000 samples per β value
- seed = 42 (reproducible)

## Dependencies

- Python 3, numpy (standard), no other dependencies
- Each β value takes ~19 seconds on a typical laptop
