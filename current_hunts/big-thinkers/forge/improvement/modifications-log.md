# Forge — Modifications Log

Record of all changes made to the Forge system (prompts, templates, orchestration, hooks). Each entry documents what changed, why, and what evidence motivated the change.

## Format

Each entry follows this structure:

```
### YYYY-MM-DD — Short description of the change

**What changed:** Which files were modified and how.

**Why:** What problem or opportunity motivated this change. Reference analysis reports or mission observations.

**Evidence:** Specific logs, metrics, or observations that justify the change. Include mission ID and task IDs where relevant.

**Risk:** Any concerns about the change or things to watch for.
```
