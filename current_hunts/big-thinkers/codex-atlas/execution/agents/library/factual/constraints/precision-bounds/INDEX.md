# Precision Experimental Bounds on Quantum Gravity

Hard quantitative bounds from experiment. Any theory making predictions outside these bounds is falsified.

Note: Several precision bounds overlap with recovery constraints (Eötvös parameter = equivalence principle, LIV time-of-flight = Lorentz invariance bounds, PPN parameters = post-Newtonian corrections). Those are filed under recovery constraints. Additional precision bounds on spacetime fluctuations and tensor-to-scalar ratio are filed under [cross-cutting/experimental-constraints.md](../../cross-cutting/experimental-constraints.md).
