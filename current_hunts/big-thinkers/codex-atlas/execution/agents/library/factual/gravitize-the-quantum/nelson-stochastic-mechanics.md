# Nelson's Stochastic Mechanics (1966)

## Core Idea

QM is equivalent to conservative diffusion (Brownian motion) in configuration space, with diffusion coefficient ℏ/2m. The Schrödinger equation IS the Kolmogorov equation for this diffusion process.

## Known Problems

- Multi-particle systems require Brownian motion in 3N-dimensional configuration space — physically opaque
- Entangled systems require non-local correlations in the stochastic process
- Nelson himself expressed doubts about the physical interpretation
- Born rule (|ψ|²) is input, not derived
- Relativistic extension fails — supraluminal influences persist
- **Multi-time correlation problem (Nelson 1966, Blanchard et al. 1986):** While single-time position probabilities agree with QM, joint probabilities for measurements at different times give wrong answers. Blanchard et al. showed this can only be fixed by assuming wave function collapse upon measurement — reintroducing the QM measurement postulate that a stochastic "derivation" of QM should avoid. This problem is inherited by any program using the Barandes-Doukas lifting (e.g., SCG), since the lifting reproduces single-time diagonal probabilities by construction but multi-time correlations require the off-diagonal elements (phases) not determined by the stochastic process alone.

## Does It "Gravitize the Quantum"?

**No.** The Brownian motion is postulated universally without gravitational motivation. However, if the noise source were identified with Planck-scale gravitational fluctuations, it becomes a potential "gravity → QM" program. This connection has been noted but not fully developed.

## Significance for the GtQ Program

Nelson provides the mathematical template for "noise → QM" programs. If the physical noise source is identified as stochastic spacetime geometry, Nelson's framework gives the mechanics of how diffusion in configuration space produces the Schrödinger equation. This makes it a key ingredient in synthesis attempts (see [`stochastic-spacetime-qm-synthesis.md`](stochastic-spacetime-qm-synthesis.md)).
