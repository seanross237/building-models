# Exploration 001 — Rindler Wedge Verification (Vacuum State)

## Mission Context

You are testing the Connes-Rovelli Thermal Time Hypothesis (TTH) in the regime it was designed for: type III₁ von Neumann algebras in QFT, where no background Hamiltonian exists and modular flow is the only candidate for time evolution.

Strategy-001 already closed off the non-relativistic equilibrium regime:
- **Normalization resolved:** τ_physical = β × t_modular (confirmed by Bisognano-Wichmann, Martinetti-Rovelli 2003 eq.18, Connes-Rovelli 1994 eq.43)
- **For Gibbs states:** K_AB = βH_AB exactly → global TTH ≡ QM. Local TTH gives single-frequency autocorrelation (no normal-mode splitting) → falsified by experiment
- **The local interpretation is what Connes-Rovelli use** (Section 4.3 of gr-qc/9406019)

This exploration is the **control test**: verifying that TTH works in its intended domain. The Bisognano-Wichmann theorem guarantees that the modular Hamiltonian of the Minkowski vacuum restricted to the right Rindler wedge is K = 2π × (boost generator). So modular flow IS the Lorentz boost. On the lattice, this should be approximately satisfied, converging to exact agreement as N → ∞.

## Specific Goal

Compute and compare two correlators for a free massless scalar field on a 1+1D lattice:

**C_local(t):** Evolution under the modular Hamiltonian of the vacuum state restricted to the right half of the lattice (the lattice Rindler wedge analogue).

**C_full(t):** Evolution under the full lattice Hamiltonian, restricted to the right sublattice.

## Detailed Setup

### Stage 1: Build the lattice scalar field

Discretize a free massless scalar field on a 1+1D lattice with N sites (start with N=100, then verify at N=50,200).

The lattice Hamiltonian for a free massless scalar field:
```
H = (1/2) Σ_i [π_i² + (φ_{i+1} - φ_i)²]
```
where π_i is the conjugate momentum at site i and φ_i is the field at site i.

This is a collection of coupled harmonic oscillators. In terms of creation/annihilation operators, diagonalize to get normal modes ω_k and the Bogoliubov transformation.

### Stage 2: Compute the vacuum correlation matrix

For the vacuum state |0⟩, compute the two-point correlation matrix restricted to the RIGHT half of the lattice (sites i = N/2+1, ..., N):

```
C_ij = ⟨0| a†_i a_j |0⟩     for i,j in the right sublattice
```

where a_i are the single-particle operators. For free fields, the vacuum is Gaussian and fully determined by this matrix.

### Stage 3: Compute the modular Hamiltonian via Peschel's formula

For free (Gaussian) fields, the modular Hamiltonian of the reduced state is quadratic:
```
K = Σ_{ij} h_{ij} a†_i a_j
```
where the single-particle modular Hamiltonian h is given by **Peschel's formula** (Peschel 2003, J. Phys. A 36 L205):
```
h = log((1 - C) / C)
```
where C is the correlation matrix from Stage 2. This is the key formula that converts the two-point function into the modular Hamiltonian.

### Stage 4: Compute both correlators

**C_local(t):** For a probe site k near the center of the right sublattice:
```
C_local(t) = Σ_{ij} [e^{iht}]_{ik} C_{kj}
```
This is the autocorrelation under modular flow. The normalization is τ_physical = (2π) × t_modular (since the "inverse temperature" for the Rindler wedge is β_Unruh = 2π).

More precisely, compute:
```
C_local(τ) = Tr[ρ_R  e^{i(h/(2π))τ}  φ_k  e^{-i(h/(2π))τ}  φ_k]
```
where the 2π normalizes modular time to physical time.

**C_full(τ):** The position autocorrelation of site k evolved under the FULL lattice Hamiltonian H (not the half-lattice H):
```
C_full(τ) = ⟨0| φ_k(τ) φ_k(0) |0⟩
```
where φ_k(τ) = e^{iHτ} φ_k e^{-iHτ} and H is the full N-site Hamiltonian.

### Stage 5: Compare and characterize

Compute:
- ||C_local - C_full|| / ||C_full|| (L² relative discrepancy)
- FFT of both to compare spectral content
- Repeat for N = 50, 100, 200 to check convergence with lattice size

## Control Checks (MANDATORY)

1. **BW profile check:** The single-particle modular Hamiltonian h should approximate the Bisognano-Wichmann profile: h_ij ∝ (distance from boundary) × (local lattice Hamiltonian density). Specifically, the diagonal of h should be approximately linear in the distance from the entangling surface (boundary of the sublattice), going to zero at the boundary and growing linearly into the bulk. Plot this profile and compare to the theoretical BW expectation.

2. **KMS check:** The reduced state ρ_R should satisfy the KMS condition at β = 2π with respect to the modular flow generated by h. Verify: for a pair of operators A, B restricted to the right sublattice, check that ⟨A(t)B⟩ = ⟨B A(t + iβ)⟩ with β = 2π. (For Gaussian states, this reduces to checking the correlation matrix transforms correctly.)

3. **Vacuum consistency:** At t = 0, C_local(0) = C_full(0) = ⟨0|φ_k² |0⟩ (both should give the same equal-time correlator).

## Success Criteria

- **Success:** C_local and C_full agree to within a few percent for N ≥ 100, with convergence improving as N increases. The BW profile check passes. This verifies TTH in its intended domain.
- **Partial success:** Agreement for bulk sites but not boundary sites (finite-size effects near the entangling surface are expected).
- **Failure:** Persistent disagreement that does NOT improve with N. If this happens, first suspect the lattice setup, not TTH — BW is a theorem, so disagreement on the lattice means the lattice approximation has a problem.

## Failure Paths

- **If C_local ≠ C_full and doesn't improve with N:** Debug the lattice setup. Check: (a) are you using the correct vacuum state? (b) is the Peschel formula h = log((1-C)/C) implemented correctly? (c) is the normalization right (modular time → physical time factor of 2π)?
- **If the BW profile check fails badly:** The lattice spacing may be too coarse. Try different boundary conditions (periodic vs. open). The BW profile is a continuum result; lattice corrections are expected but should be small for sites far from the boundary.
- **If memory/time issues for large N:** N=100 gives a 50×50 correlation matrix for the half-lattice, which is easily tractable. If N=200 is too slow, report the N=50,100 results.
