# Exploration 007: Devil's Advocate — Attack Causal Fakeon Theory

## Mission Context

Exploration 006 built Causal Fakeon Theory (CFT) — 4 axioms + 1 conjecture connecting causal partial order to the fakeon prescription. The strategy's #1 meta-lesson from SCG: "attack the theory IMMEDIATELY after construction, not after multiple development rounds."

## CFT's Claims (What You're Attacking)

**Axiom 1:** Spacetime is fundamentally a causal set. Causal order determines conformal geometry (Malament), volume measure determines conformal factor (Sorkin).

**Axiom 2:** The SJ vacuum construction determines propagators from causal structure. Modes with negative spectral weight break the SJ construction and cannot be physical particles.

**Axiom 3:** Excluded modes propagate via the Wheeler propagator G_W = (G_R + G_A)/2 = fakeon propagator at tree level. This is independently proven by Belenchia et al. (2015) for causal set unstable modes.

**Axiom 4:** Perturbative self-consistency — the fakeon doesn't invalidate the background causal order because it only contributes virtually.

**Conjecture:** The tree-level derivation extends to loop level, reproducing Anselmi's full fakeon prescription.

**Novel predictions:** (1) Dimensional crossover profile at fakeon mass, (2) stochastic microcausality violation, (3) quadratic gravity should emerge from causal set actions.

## Your Task: Try to BREAK It

Use the following attack vectors. For each, rate severity: FATAL / NEAR-FATAL / SERIOUS / MODERATE / MINOR.

### Attack Vector 1: The Loop-Level Gap
- Anselmi (2020) explicitly showed the Wheeler propagator fails at loop level. CFT's conjecture that the causal set formalism can reproduce the full fakeon prescription is UNPROVEN.
- Search: Is there ANY progress on interacting QFT on causal sets? What is the state of the SJ construction for interacting theories?
- Can you find a specific argument showing the loop extension is IMPOSSIBLE (not just unproven)?

### Attack Vector 2: The SJ Construction Limitations
- The SJ construction exists only for free scalar fields on fixed backgrounds.
- The ghost in QG+F is spin-2, not scalar. Can SJ work for higher-spin fields?
- In quantum gravity, the background is dynamical. Can SJ work on dynamical backgrounds?
- Search: What is the current state of the SJ program? Has it been extended beyond free scalars?

### Attack Vector 3: Circularity
- CFT uses causal structure to constrain the spin-2 ghost, but the spin-2 ghost is a mode of the metric that DEFINES causal structure.
- Is this a genuine logical circularity, or is it resolved by perturbation theory (background causal order is fixed; ghost is a perturbation)?
- Compare to similar circularities in other QG programs. Is CFT's circularity worse or better?

### Attack Vector 4: Does This Actually Derive the Fakeon?
- CFT's "derivation" goes: causal set → SJ construction → negative spectral weight excluded → Wheeler propagator. But you need to KNOW the spectral weight is negative before applying the exclusion.
- If you didn't already know the ghost has negative residue, would CFT tell you?
- Is CFT genuinely predictive (tells you which modes are fakeons) or just explanatory (gives a nice story for why the ghost is a fakeon)?

### Attack Vector 5: Are the Novel Predictions Real?
- Prediction 1 (dimensional crossover): Is this distinguishable from standard QG+F's spectral dimension running?
- Prediction 2 (stochastic microcausality): Is this different from Anselmi's existing microcausality violation?
- Prediction 3 (quadratic gravity from causal sets): Is there any reason to expect this? Causal set actions reproduce EH, not quadratic gravity.
- Are ANY of CFT's "novel predictions" actually new, or are they reformulations of existing QG+F predictions?

### Attack Vector 6: Comparison to Existing Programs
- Is CFT actually novel, or is it just "causal set theory + quadratic gravity" repackaged?
- What does a causal set theorist (Sorkin, Dowker) gain from CFT that they don't already have?
- What does Anselmi gain from CFT that he doesn't already have?
