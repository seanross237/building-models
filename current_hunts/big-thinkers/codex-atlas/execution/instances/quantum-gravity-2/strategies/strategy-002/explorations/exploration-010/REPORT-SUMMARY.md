# Exploration 010 Summary: Future Directions Survey

## Goal
Survey 4 directions for the next strategy: (A) non-perturbative QG+F / ghost confinement, (B) parameter determination / observables, (C) new mathematics, (D) QG+F–AS connection.

## What Was Done
Web research on all four directions, covering: lattice QG simulations, CDT phase structure, ghost confinement analogy, Donoghue's physical running results (2024), the new IHO/direct-sum QFT reinterpretation (March 2026), LiteBIRD projections, gravitational wave QNM calculations in modified gravity, functorial QFT, information-theoretic gravity, AS-vs-QG+F status in the literature, and Donoghue's critique of AS.

## Outcome: Succeeded

## Key Findings

1. **No one has simulated QG+F on a lattice.** CDT's continuum limit is likely Hořava-Lifshitz, not QG+F. Ghost confinement remains a pure conjecture with no computational path to test it.

2. **LiteBIRD (launch ~2032, δr < 0.001)** will sharply test QG+F inflation models. Gravitational wave QNM corrections from quadratic gravity are suppressed by (M_BH/M₂)² — unmeasurably small for astrophysical BHs if M₂ ~ M_Planck.

3. **QG+F and AS are explicitly treated as distinct approaches** in the 2025 literature. QG+F has asymptotic freedom (couplings → 0); AS has an interacting fixed point (couplings → finite). This is a sharp, well-defined question that existing tools can address.

4. **A March 2026 paper** reinterprets the ghost as an IHO instability in "direct-sum QFT" — genuinely new mathematics, but too early to build a strategy around.

## Top Recommendation
**Direction D: QG+F–AS Connection** — best feasibility/impact ratio. The question is sharp, tools exist, and the answer reshapes the field. Combine with Direction B (compute observational discriminators). Avoid Directions A and C as primary targets (too speculative / too slow).
