# Exploration 003: Minimal Falsification Model for Harmonic Far-Field Pressure

## Goal

Construct the smallest explicit model that tests the only surviving question:

Can a harmonic far-field pressure component on the local cylinder make the bad pressure pairing genuinely `U_k`-dependent, or does harmonicity only change constants / oscillation while leaving the coefficient fixed at energy scale?

This is a negative-model exploration unless the model shows a real structural gain.

## Preloaded context

- Exploration 001: the literal Vasseur bottleneck is local `P_{k21}`; the loophole is an alternative near/far decomposition question.
- Exploration 002: the loophole is Tao-incompatible, so the Tao filter does not close the route.
- Local library now contains:
  - `exact-far-field-pressure-obstruction.md`
  - `h1-pressure-dead-end.md`
  - `non-cz-pressure-routes-tool-independence.md`

## Required deliverables

1. An explicit local model with:
   - a ball or cylinder `Q`,
   - a decomposition `p = p_near + p_far`,
   - `Δp_far = 0` on `Q`,
   - `p_far` generated by data outside `Q`.
2. A concrete pressure pairing to test, ideally mirroring the mission language:
   - `∫∫_Q p_far ψ dx dt`
   or an equally explicit local test-function pairing.
3. A sharp answer to whether harmonicity alone can force that pairing to carry a positive power of the local De Giorgi quantity `U`.
4. If the answer is no, state the exact obstruction:
   - dependence only on exterior/energy-scale data,
   - constant or affine harmonic modes surviving,
   - Harnack/oscillation control improving regularity but not scaling,
   - or equivalent.
5. A verdict:
   - `harmonicity does not create U-dependence`
   - or `harmonicity creates a real U-dependent gain`

## Preferred model forms

Use the simplest explicit construction that settles the issue. Examples:

- exterior source producing a harmonic Newtonian potential inside a smaller ball,
- constant / affine harmonic modes on the local ball,
- Wolf-style local harmonic+particular split specialized to the simplest geometry.

If one model already closes the question cleanly, stop there.

## Constraints

- Do not build an aspirational model with unspecified exterior data.
- Do not drift back into Tao or a broad pressure survey.
- Keep the comparison tied to the actual mission target: does the coefficient become `U`-dependent or stay fixed?

## Output format

Write:

1. `REPORT-SUMMARY.md`
2. `REPORT.md`

Structure `REPORT.md` as:

1. Executive verdict
2. Explicit model
3. Pairing computation
4. Why harmonicity does or does not change scaling
5. Consequence for the mission

Tag substantial statements as `[VERIFIED]`, `[CHECKED]`, or `[CONJECTURED]`.
