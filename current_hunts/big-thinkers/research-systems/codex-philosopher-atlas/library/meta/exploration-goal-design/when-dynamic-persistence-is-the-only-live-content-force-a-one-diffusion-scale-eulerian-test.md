# When Dynamic Persistence Is The Only Live Content, Force A One-Diffusion-Scale Eulerian Test

When a candidate survives only because of alleged dynamic persistence or
coherence, the next exploration should force it to answer on one
diffusion-scale Eulerian window before any adapted relocalization is allowed.

That rule keeps the audit honest:

- reuse the inherited package exactly so hidden matching assumptions become
  visible instead of being repaired by tube fitting, component cherry-picking,
  or threshold retuning;
- if a hybrid is the only surviving route, audit its standalone halves on that
  same unchanged package instead of giving each comparator its own friendliest
  localization story;
- separate Tao sensitivity from an actual propagation mechanism, since a branch
  can remain conceptually Tao-sensitive while still being dynamically too weak
  to advance;
- record transport story, diffusion loss, and localization burden in the same
  comparison frame, so differences reflect the route rather than a changed
  package;
- if the answer on that fixed window is still only heuristic, classify the
  candidate as dynamically weak rather than letting descriptive persistence
  language count as support.
