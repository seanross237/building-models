# When An Intrinsic Phase Object Is The Goal Promote The Orbit Ledger Before Scalar Scores

When a mission asks for an intrinsic phase/coherence object, first promote the
orbit-level exact interaction ledger.
For phase branches, that means coefficient-corrected triad-orbit data or
another symmetry-respecting exact interaction sheet, not the first convenient
scalar summary.

Scalar coherence scores should stay secondary unless the aggregation law
itself is forced by the same symmetry class.
Weighted averages, entropy scores, defect medians, window thresholds, and
similar compressions may be useful diagnostics, but they are policy choices
whenever several equally natural summaries live on the same exact ledger.

Make the inadmissibility rule explicit in the same prompt:
if the raw formula names desired routes, spectator sets, packet roles, or any
hoped-for transfer direction by hand, the branch has already left the
intrinsic-object screen.

This sequencing prevents an exploration from promoting a tidy scorecard as if
it were the theorem-facing object and keeps later diagnostics anchored to
exact interaction data rather than to target labels.
