# A Scope Lock Is Not Real Until Baseline Cash-Out And Threshold Are Frozen In The Same Burden Currency

When a branch is sold as an early setup gate for a later coercivity test, do
not count the scope lock as real just because the endpoint family, invariant
package, and Tao-sensitive feature have been narrowed.

The setup is still obstructed until the record freezes both:

- one explicit baseline inequality, exclusion statement, or contradiction line;
- one non-cosmetic gain threshold or admissible gain notion that later steps
  must beat;

and it freezes them in the same burden currency that the later coercivity test
will use.

Packet-level Tao sensitivity can become concrete before coercivity exists, but
that does not rescue a branch whose baseline and threshold remain at slogan
level.

If later debt is charged back in some different ledger, the setup has reverted
to narrative bookkeeping rather than a real mechanical scope lock.
