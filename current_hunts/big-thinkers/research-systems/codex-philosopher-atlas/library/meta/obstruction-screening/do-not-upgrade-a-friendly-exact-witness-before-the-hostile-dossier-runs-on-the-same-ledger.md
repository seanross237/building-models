# Do Not Upgrade A Friendly Exact Witness Before The Hostile Dossier Runs On The Same Ledger

A friendly exact packet family may show that a construction-facing mechanism is
realizable, but that witness is still only class-local until the hostile
dossier is run on the same frozen ledger.

Do not promote the friendly witness into theorem evidence unless the friendly
and hostile comparisons use the same packet object, window, thresholds,
spectator partition, sign sheet, and burden currency.

Otherwise the branch mistakes a one-sided construction example for a symmetric
discriminator and upgrades a local positive witness before the actual
comparison protocol has been completed.
