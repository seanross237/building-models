# When A Candidate Only Survives By Architecture Drift, Count That As Failure

In fixed-obstruction audits, one of the fastest discriminator questions is:
is this candidate still acting on the frozen bad term, or has it quietly become
a different branch?

If the only apparent rescue is to shift from a variable-choice rewrite into
stretching, geometry, or some other rival architecture, that is not a near
miss. It is evidence against continuation of the original candidate.

This keeps Navier-Stokes-looking reformulations honest. Variable choice is not
yet a discriminator unless the same fixed estimate improves without changing
the branch being audited.
