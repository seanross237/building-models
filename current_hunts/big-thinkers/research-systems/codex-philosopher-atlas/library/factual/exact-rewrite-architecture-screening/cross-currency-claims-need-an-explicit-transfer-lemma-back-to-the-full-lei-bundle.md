# Cross-Currency Claims Need An Explicit Transfer Lemma Back To The Full LEI Bundle

Status: `INFERRED` transfer rule with `VERIFIED` scope-lock support

When a claimed gain is stated in a different currency from the frozen LEI
burden, it does not count on its own.

This includes derivative-count reductions, integrability improvements,
surrogate cancellations, and compactness-side leverage.

Each such claim needs an explicit transfer lemma that names how the surrogate
gain yields a positive-margin smaller effective coefficient on the full
localized `I_flux[phi]` bundle, under the same solution class and shared
cutoff protocol, after all new debt is charged.

If no explicit transfer is written, the claim remains unresolved and does not
count as progress for the branch.
