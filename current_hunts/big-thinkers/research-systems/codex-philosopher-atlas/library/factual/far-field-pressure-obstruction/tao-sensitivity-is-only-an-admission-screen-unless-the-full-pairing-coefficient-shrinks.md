# Tao Sensitivity Is Only An Admission Screen Unless The Full Pairing Coefficient Shrinks

Status: `VERIFIED` with `INFERRED` branch rule

For this far-field pressure branch, averaged-model sensitivity is mandatory but
insufficient.

Each later mechanism must pass two tests:

- the relevant feature must fail the appropriate averaged-model comparison, or
  else it is dead for the branch;
- but even genuine Tao sensitivity counts only if the same candidate also
  lowers the effective coefficient in the frozen full-pairing estimate for
  `I_p^far` after reconstruction and localization debt are restored.

So Tao sensitivity is only an admission screen. On this ledger it does not
license continuation by itself, and it does not let a candidate survive by
using Navier-Stokes-specific language as leverage while the theorem-level
far-field estimate stays unchanged.
