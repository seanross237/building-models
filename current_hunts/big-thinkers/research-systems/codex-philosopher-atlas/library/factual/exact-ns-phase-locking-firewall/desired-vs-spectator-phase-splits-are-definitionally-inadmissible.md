# Desired-Vs-Spectator Phase Splits Are Definitionally Inadmissible

Status: `VERIFIED` mission inadmissibility rule with `INFERRED`
phase-object application

Any raw phase score that splits active triads into a `desired-transfer` set
and a `spectator` set is definitionally inadmissible on this branch.

Once the formula names those subsets by hand, the object no longer comes from
intrinsic exact interaction data.
It only exists after importing the hoped-for transfer route.
That remains true even if the resulting scalar can be normalized canonically
on a fixed sheet.

The same rejection applies to raw reuse of predecessor packet-role labels or
Step-6 gate labels inside the definition of the phase object.
