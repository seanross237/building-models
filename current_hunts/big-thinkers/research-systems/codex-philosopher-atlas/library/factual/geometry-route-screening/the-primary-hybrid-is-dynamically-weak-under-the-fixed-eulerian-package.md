# The Primary Hybrid Is Dynamically Weak Under The Fixed Eulerian Package

Status: `INFERRED` with `VERIFIED` support

The Step-4 dynamic screen leaves the primary geometry survivor
`direction coherence + tube persistence` alive only as
`informative but dynamically weak` when it is forced to answer inside the
fixed Eulerian package
`B_r(x_*) x [t_* - r^2, t_*]` with `|omega| >= r^-2`.

The reusable reason is structural:

- the local record still offers only a heuristic transport story for coherent
  tube persistence across the full diffusion-scale window, not a theorem-level
  propagation or matching principle inside the moving intense set;
- because the package is neutral and Eulerian rather than tube-adapted, the
  burden is to match coherent intense components across time without
  re-centering, axis fitting, threshold retuning, or other hidden adaptation;
- diffusion and localization-evolution costs act at order one on the chosen
  `r^2` window, so the hybrid's Tao-sensitive content does not by itself
  supply a robust dynamic mechanism.

Operationally, later steps may still treat the hybrid as the branch's best
conceptual survivor, but not as dynamically supported unless a concrete
propagation or Eulerian matching principle is supplied inside the audited
package.
