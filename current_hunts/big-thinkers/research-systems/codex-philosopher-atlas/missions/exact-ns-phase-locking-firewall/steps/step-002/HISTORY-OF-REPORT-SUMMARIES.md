## Exploration 001 Summary

- Goal:
  classify the canonical first-budget seed families for one exact triad orbit
  on the frozen canonical helical support sheet.
- Outcome:
  `succeeded`.
- Takeaway:
  `[INFERRED]` every honest first-budget seed is a nondegenerate one-triad
  helical seed modulo conjugation, `p <-> q`, and global helical-sign
  relabeling, and the raw sign triples reduce to three canonical classes:
  `(+++)`,
  `(++-)`,
  `(+--)`.
  Repeated-wavevector and collinear cases are excluded because they are not
  exact active triads with nonzero projected coefficient.

## Exploration 002 Summary

- Goal:
  run the recursive exact closure audit on each canonical one-triad
  first-budget seed.
- Outcome:
  `succeeded`.
- Takeaway:
  `[INFERRED]` no honest first-budget seed survives.
  Once the one-triad seed is mirror-completed, off-orbit active pairs such as
  `(p, -q)`, `(k, p)`, and `(k, q)` force new wavevectors
  `p - q`,
  `2p + q`,
  and
  `p + 2q`,
  so every canonical sign-sheet class dies by the same current-budget
  obstruction:
  recursive closure spills beyond the one-triad budget on the first honest
  pass.

## Exploration 003 Summary

- Goal:
  run the required admissible-enlargement audit on any apparent first-budget
  survivor, or package a clean current-budget negative if no honest survivor
  remains.
- Outcome:
  `succeeded`.
- Takeaway:
  `[INFERRED]` the enlargement audit confirms the negative.
  Adding the first forced shared-mode orbit on `p - q` immediately forces
  further new modes such as
  `2p - q`,
  `-2q`,
  `2p`,
  and
  `p + 2q`,
  so the mirror-complete one-triad picture is only a pseudo-survivor created
  by over-pruning.
  No genuinely invariant finite support survives the first budget, and Chain
  Step 3 is not earned on this rung.
