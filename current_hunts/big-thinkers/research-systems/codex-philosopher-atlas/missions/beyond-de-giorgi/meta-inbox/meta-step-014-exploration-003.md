# Meta Note - Step 014 Exploration 003

- For reformulation-first audits, the first real kill gate is often endpoint
  exchange, not formal anti-repackaging classification.
- If the theorem-facing currency is fixed, require the transfer lemma to be
  stated explicitly; "the native variable looks cleaner" is not an exchange
  path.
- Prior-art overlap still matters, but if no same-currency cash-out exists even
  before overlap screening, record exchange failure as the primary obstruction.
