## Findings

### 1. Admissibility decision

- [VERIFIED] The only locally frozen theorem floor remains the inherited
  suitable-weak / Leray-Hopf plus LEI framework.
- [VERIFIED] The geometry record does not leave a directly theorem-floor-native
  positive mechanism on that floor.
  Direction coherence remains stretching-facing but dynamically weak.
  Tube persistence remains live only as a dynamic or hybrid route and collapses
  under fixed Eulerian auditing without stronger tracking.
- [INFERRED] The live candidate from Exploration 001 is therefore
  `proxy-based`, not directly theorem-floor admissible.
- [INFERRED] The named proxy class should be frozen as:
  `neutral-Eulerian intense-set coherence/persistence class`,
  meaning the high-vorticity proxy built from the thresholded intense set and a
  hoped-for coherent tube family with persistent direction coherence across the
  diffusion-scale window.

### 2. Exact bridge burden back to the theorem floor

- [INFERRED] The bridge burden has three inseparable parts:
  1. prove that coherent intense components can be matched across the full
     `r^2` window inside the neutral Eulerian package without tube-adapted
     relocalization, threshold retuning, or hidden axis fitting;
  2. propagate direction coherence on that moving intense set without importing
     the stronger `nabla xi`-type derivative control that the local record
     treats as a return to known direction-regularity burden; and
  3. convert that proxy persistence/coherence information into a smaller
     criterion-scale input or a smaller full-stretching contribution on the
     inherited theorem floor.
- [VERIFIED] The local record already identifies exactly these burdens as the
  missing mechanism behind the hybrid's failure:
  Eulerian component matching,
  diffusion-scale persistence,
  hidden relocalization,
  and `nabla xi`-type debt.
- [INFERRED] This bridge does not look cheaper than the target theorem.
  It already looks `theorem-scale`.

### 3. Primary comparator

- [VERIFIED] The closest live geometric prior-art comparator is the
  `Constantin-Fefferman / direction-regularity` neighborhood.
- [VERIFIED] Shared structure already present there:
  vorticity-direction coherence as the theorem-facing control language.
- [INFERRED] Claimed mathematical delta of the proposed route:
  add an NS-specific dynamic persistence ingredient, via coherent tube
  persistence on intense regions, so that the direction-coherence input would
  no longer be merely a static criterion hypothesis but a dynamically produced

### 4. Comparator absorption check

- [VERIFIED] The repository already warns that direction regularity is prior
  art and that importing stronger `nabla xi`-type control just collapses the
  route back into that comparator family.
- [VERIFIED] The repository also warns that tube persistence without an honest
  dynamic bridge collapses to descriptive geometry.
- [INFERRED] So the proposed delta is very narrow even before package choice:
  it survives only if the hybrid bridge is both theorem-floor cheaper than the
  target theorem and genuinely dynamic.
- [INFERRED] The local dynamic audits instead say the hybrid is only
  `informative but dynamically weak`, while all audited candidates fail before
  later representation work.

### 5. Earliest honest obstruction

  `theorem-floor obstruction, branch should stop before package selection`.
- [INFERRED] Reason:
  a singular endpoint family, gain currency, and comparator can be frozen, and

- [INFERRED] The neutral Eulerian package remains useful only as a killed
