- Goal:
  define the bounded Step-3 observable table for the fixed package
  `filament or tube concentration` versus `sheet or pancake concentration`
  under the Eulerian localization
  `[PROPOSED] |omega| >= r^{-2}` on
  `[PROPOSED] B_r(x_*) x [t_* - r^2, t_*]`.

- What was checked:
  - inherited Step-2 survivor roles and Tao-screen classifications
  - Step-3 scenario decision from Exploration 001
  - Step-3 localization decision from Exploration 002
  - the geometry-screen rule that every candidate must name a future
    full-stretching-facing leverage point and a Tao discriminator
  - whether any candidate needed a new hidden normalization beyond the fixed
    Eulerian package

- Outcome:
  `succeeded`

- One key takeaway:
  the fixed Step-3 package keeps exactly one honest primary input alive,
  `direction coherence + tube persistence`; everything else is either a
  comparator or a fragility screen.

- Compact candidate ranking:
  - primary:
    `direction coherence + tube persistence`
  - secondary:
    `vorticity-direction coherence`
    `tube coherence / persistence`
  - effectively dead as promoted routes, retained only as fragility screens:
    `local Beltrami / alignment`
    `Beltrami deficit + concentration`
    `Beltrami deficit + anisotropy`

- Leads worth pursuing:
  - in the next dynamic plausibility screen, ask what quantity is actually
    propagated on the Eulerian intense set and whether that propagation can
    shrink either the exterior/inter-scale stretching contribution or the
    localization-evolution cost
  - use `vorticity-direction coherence` to test whether the hybrid adds
    anything beyond prior-art direction regularity
  - use `tube coherence / persistence` to test whether the tube component has
    any stretching-facing content once the direction factor is removed

- Unexpected findings:
  - the fixed Eulerian package does not kill the primary hybrid, but it strips
    away most of the rhetoric that had made the branch look broader
  - no Beltrami-based candidate was rescued by the honest scenario/localization
    choices; they remain diagnostic only

- Computations worth doing later if still in scope:
  - formalize a persistence score on `E_r(t)` that does not re-center the ball,
    re-fit a tube axis, or normalize by a local maximum
  - test modest perturbations of the threshold constant and window length to
    see whether the primary hybrid is stable or merely tuned

- Advance recommendation:
  advance to the dynamic plausibility screen, but only narrowly and only with
  the fixed Step-3 package. Kill the branch immediately if the primary hybrid
  needs tube-adapted relocalization, tuned thresholds/windows, or stronger
  direction-derivative control than the bounded package contains.
