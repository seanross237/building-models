# Exploration 001 Goal — Dynamic Screen For The Primary Hybrid

## Objective

Test the primary Step-3 survivor

`direction coherence + tube persistence`

against the fixed Step-3 package inherited from `step-003`:

- primary scenario: `filament or tube concentration`
- comparator scenario retained only for context: `sheet or pancake concentration`
- localization: Eulerian parabolic package on
  `B_r(x_*) x [t_* - r^2, t_*]`
- threshold: `|omega| >= r^{-2}`

This exploration must decide whether the primary hybrid has a credible dynamic
story under that fixed package, rather than only a static geometric picture.

The report must identify, as concretely as the local repository allows:

- the best available transport identity or approximate propagation heuristic;
- the main diffusion loss for coherence or tube persistence over one
  `r^2`-scale window;
- the main commutator, cutoff, or localization-evolution burden introduced by
  using the fixed Eulerian package instead of a tube-adapted one;
- what part of the fixed package the story depends on most;
- whether the Tao discriminator remains live dynamically;
- and the first honest failure mode.

If the local repository does not support a credible dynamic persistence story
without hidden tube adaptation or stronger derivative control, say so directly
and classify the route accordingly.

## Success Criteria

- Produces a decisive dynamic memo, not generic optimism.
- Uses `[VERIFIED]`, `[INFERRED]`, and `[PROPOSED]` labels consistently.
- Names exact source files for every major transport, diffusion, or burden
  claim.
- Keeps the argument bounded to the fixed Step-3 package instead of redesigning
  localization.
- States explicitly whether the Tao discriminator remains live once persistence
  is phrased dynamically rather than descriptively.
- Ends with:
  - best transport / propagation story
  - main diffusion loss
  - main localization-evolution or commutator burden
  - first obvious dynamic failure mode
  - final rating:
    `dynamically plausible`, `informative but dynamically weak`, or
    `static-only`

## Failure Criteria

- Treats tube imagery as a dynamic mechanism.
- Reintroduces tube-adapted coordinates, tube-fitted axes, or re-centered
  threshold choices without naming them as new assumptions.
- Imports a stronger `nabla xi`-type or derivative-control package without
  acknowledging that it exceeds the inherited Step-3 bounds.
- Slides into kernel-level `S omega . omega` decomposition instead of staying at
  the dynamic-screen level.
