# Exploration 002 Goal - Attach Exact Interaction Templates And Downstream Gates

## Objective

For each promoted candidate notion from Exploration 001, write the exact
Fourier / helical interaction template it is supposed to track and assign one
honest downstream gate.

## Success Criteria

- For each promoted candidate, name the desired role couplings, the forced
  same-scale / conjugate / cross-scale spectators, and the exact data later
  steps will still need.
- Keep the template concrete enough that Step 3 can run the robustness audit
  and Step 4 can prepare normalized exact-packet tests.
- Give each promoted candidate exactly one downstream gate:
  either one invariant dynamical observable on one stated packet family and
  finite window, or one invariant admissibility sheet for adversarial exact
  search.
- Decide whether Chain Step 3 is now well-posed.

## Failure Criteria

- Any promoted candidate still lacks an exact interaction object.
- Any promoted candidate still lacks one honest downstream gate.
- The promoted family collapses into a mechanism memo instead of a real object.
