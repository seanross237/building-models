# Exploration 006: Deep Build — Causal Fakeon Theory (CFT)

## Goal

Build a focused theory — Causal Fakeon Theory (CFT) — connecting the fundamental causal partial order of spacetime to the fakeon prescription in quadratic gravity. State axioms precisely, derive physical consequences, address the loop-level problem honestly, compare to standard QG+F, and identify at least one novel prediction.

---

## 1. Axioms and Principles

CFT rests on four axioms and one conjecture. The axioms are ordered from most established to most speculative.

### Axiom 1 — Causal Foundation

**Physical spacetime is fundamentally a locally finite partial order (causal set) C = (𝒞, ≺).** The partial order ≺ encodes causal precedence: x ≺ y means x can influence y. The partial order determines the conformal geometry of the continuum approximation (Malament's theorem, 1977: any bijection preserving the chronological relation between two future/past-distinguishing spacetimes is a conformal isometry). A discrete volume measure (element counting) determines the conformal factor (Sorkin's "order + number = geometry").

**Status:** Established mathematics. Malament's theorem is proven. The causal set program (Bombelli-Lee-Meyer-Sorkin, 1987) implements this as a quantum gravity approach. The Benincasa-Dowker action recovers the Einstein-Hilbert action in the continuum limit.

### Axiom 2 — Spectral Causal Compatibility

**The vacuum state and propagators of every quantum field on C are determined by the causal structure via the Sorkin-Johnston (SJ) construction.** Specifically:

1. The commutator function (Pauli-Jordan function) Δ(x,y) = [ϕ(x), ϕ(y)] is computed from the retarded Green function, which depends only on the causal order.
2. The SJ vacuum is defined by the positive part of the spectrum of iΔ. Eigenvectors with positive eigenvalues define "positive frequency modes"; negative eigenvalues define "negative frequency modes."
3. The Wightman two-point function (and hence the propagator) is constructed from the positive-eigenvalue sector alone.

**The causal compatibility condition:** For a mode with mass m and residue r at the pole p² = m²:
- If r > 0 (positive spectral weight): The SJ construction yields a well-defined Fock vacuum. The mode propagates via the Feynman propagator G_F, with on-shell particle production permitted.
- If r < 0 (negative spectral weight): The SJ construction yields negative-norm states — the "positive frequency" decomposition produces a state space incompatible with probabilistic interpretation. **The causal order cannot support this mode as a physical particle.**

**Status:** The SJ construction is rigorously defined for free scalar fields on globally hyperbolic spacetimes and on causal sets (Johnston 2009, Afshordi-Aslanbeigi-Sorkin 2012). The extension to negative-spectral-weight modes is the novel element of CFT — the claim that the SJ construction's breakdown for ghost modes is not a bug but the physical mechanism that enforces the fakeon prescription.

### Axiom 3 — Wheeler Exclusion

**Modes excluded from the physical spectrum by Axiom 2 propagate via the Wheeler propagator G_W = (G_R + G_A)/2, which is the unique ≺-compatible treatment for non-physical modes.** This follows from:

1. The Wheeler propagator is the real part of the Feynman propagator: G_W = Re(G_F) = P.V. 1/(p² − m²). It has no on-shell poles — no particle production.
2. In causal set QFT, Belenchia, Benincasa & Liberati (JHEP 03(2015)036) proved that the nonlocal d'Alembertian derived from causal set dynamics produces precisely the Wheeler propagator for unstable modes in 4D. These modes "do not appear in the asymptotic states."
3. The Wheeler propagator is symmetric under time reversal: G_W(x,y) = G_W(y,x). A mode with negative residue has anti-causal energy flow (Donoghue 2019: ghost particles propagate backwards in time). The only ≺-compatible propagator for such a mode is one with no preferred temporal direction — i.e., the Wheeler propagator.

**The mathematical identity:** At tree level, the fakeon propagator (Cauchy principal value prescription) is exactly the Wheeler propagator:

    G_fakeon(p) = P.V. 1/(p² − m²) = (G_R(p) + G_A(p))/2 = G_W(p)

**Status:** The tree-level identity G_fakeon = G_W is exact and recognized by Anselmi (JHEP 03(2020)142). The Belenchia et al. result is proven for scalar fields on causal sets. The novel claim of CFT is that these two independent results have a common origin: the causal partial order.

### Axiom 4 — Perturbative Self-Consistency

**The causal order ≺ is dynamically generated by the theory itself: the metric g_μν (which defines ≺ in the continuum limit) is a solution of the quadratic gravity field equations. The fakeon mode χ_μν (massive spin-2 perturbation) is a perturbation of ≺. Self-consistency requires:**

1. The background causal order ≺₀ (defined by the background metric ḡ_μν) determines the propagator prescriptions for all perturbative modes via Axioms 2-3.
2. The fakeon χ_μν contributes to the full causal order only through virtual (loop) effects — never as an on-shell disturbance that could modify ≺₀.
3. This is self-consistent precisely because the fakeon is purely virtual: it corrects the background geometry at the quantum level (renormalization of Newton's constant, running couplings) but never produces a classical on-shell perturbation that would invalidate the background causal structure used to define it.

**Status:** This is the weakest axiom. It is valid within perturbation theory around a fixed background, where it functions as a consistency check. Beyond perturbation theory, the circularity is unresolved — see Section 3.

### Conjecture — Loop Extension (The Central Open Problem)

**The tree-level Wheeler prescription extends to a full loop-level prescription that coincides with Anselmi's fakeon prescription at every order in perturbation theory.** Specifically:

At loop level, the fakeon prescription is NOT simply "use G_W in all internal lines." Anselmi (JHEP 03(2020)142) proved that naive use of the Wheeler propagator in loops violates renormalizability, unitarity, and stability (due to coexistence of +iε and −iε). The correct fakeon procedure (Anselmi, JHEP 11(2021)030) is:

1. Compute loop diagrams using the standard Feynman prescription for all propagators.
2. Apply the spectral optical theorem to decompose amplitudes into (multi)threshold contributions.
3. **Drop all thresholds that involve fakeon frequencies** — remove them from the spectral decomposition.
4. The remaining amplitude satisfies unitarity (optical theorem) by construction.

**CFT's conjecture:** This loop-level procedure has a causal-order interpretation. Dropping fakeon thresholds corresponds to enforcing, at every loop order, the requirement that the ghost mode cannot go on-shell — i.e., the causal order forbids the ghost from materializing at any order in perturbation theory, not just at tree level. The SJ construction, if extended to interacting fields on causal sets, should reproduce this threshold-dropping procedure.

**Status:** Unproven. This is the make-or-break challenge for CFT. See Section 3 for detailed analysis.

## 2. Physical Consequences

### 2.1 Recovery of QG+F at Tree Level

CFT reproduces standard QG+F at tree level by construction. The quadratic gravity action (Stelle 1977) gives the propagator:

    G_2(p²) = 1/p² − 1/(p² − M₂²)

The massless graviton (positive residue at p² = 0) receives the Feynman prescription via Axiom 2. The massive spin-2 mode (negative residue −1 at p² = M₂²) receives the Wheeler/fakeon prescription via Axiom 3. The massive scalar (positive residue at p² = M₀²) receives the Feynman prescription (or optionally fakeon, if one imposes additional constraints). Tree-level scattering amplitudes are identical to QG+F.

### 2.2 Newton's Law and GR Limit

At distances r ≫ 1/M₂, 1/M₀, the massive modes decouple exponentially. The Stelle potential:

    V(r) = −GM/r × [1 + (1/3)e^{−M₀r} − (4/3)e^{−M₂r}]

reduces to V(r) → −GM/r. All GR tests (PPN parameters, GW speed, Lorentz invariance) are satisfied identically to QG+F. CFT adds no IR modifications.

### 2.3 Microcausality Violation as Causal Granularity

In standard QG+F, microcausality violation (commutators non-zero at spacelike separation below ~1/M₂) is treated as an acceptable price for renormalizability + unitarity. Anselmi (2026) goes further, arguing causality should be "abandoned in fundamental physics."

**CFT offers a different interpretation:** The microcausality violation is not a sacrifice — it is a **signature of the discrete causal order**. Below the discreteness scale ℓ ~ 1/M₂, the partial order ≺ is not well-defined: events are neither causally related nor spacelike-separated but simply *unresolved*. Field commutators fail to vanish at spacelike separation below ℓ because the concept of "spacelike" loses meaning at the discreteness scale.

This reframes the violation from a cost (Anselmi's view) to a prediction: **microcausality violation is the continuum signature of causal set discreteness**. The violation scale 1/M₂ is identified with the causal set's fundamental discreteness scale.

### 2.4 Spectral Dimension Flow d_s = 4 → 2

QG+F predicts d_s = 2 in the UV from its propagator structure (~1/p⁴ at high momenta). Independently, causal set dynamics also produce d_s → 2 at short distances. Belenchia et al. (Phys. Rev. D 93, 044017, 2016) showed that nonlocal d'Alembertians derived from causal sets exhibit **universal dimensional reduction to d_s = 2 in all spacetime dimensions**.

**CFT unifies these:** Both reductions have the same origin — the causal partial order. The 1/p⁴ propagator of quadratic gravity is the continuum expression of the nonlocal causal set d'Alembertian. The d_s = 2 reduction is not a coincidence across approaches but a consequence of the shared causal foundation. CFT predicts the crossover scale is M₂ (the fakeon mass): d_s = 4 for E ≪ M₂, flowing to d_s = 2 for E ≫ M₂.

### 2.5 Analyticity Sacrifice Explained

In standard QG+F, the sacrifice of S-matrix analyticity is an unexplained cost — Anselmi's 2025 classification (arXiv:2503.01841) shows it is forced by demanding unitarity + Lorentz invariance, but provides no deeper reason.

**CFT offers an explanation:** Analyticity of the S-matrix requires that all intermediate states in scattering be physical (the Cutkosky rules connect imaginary parts of amplitudes to on-shell intermediate states). The fakeon, being purely virtual, removes certain intermediate states from the physical spectrum. The analyticity failure is the S-matrix's way of encoding the fact that the causal order forbids certain modes from materializing — the "missing" analytic structure corresponds to the "missing" on-shell fakeon states.

This is a conceptual explanation, not a derivation. But it transforms the analyticity sacrifice from an unmotivated cost to a consequence of causal-order enforcement.

## 3. The Loop-Level Problem

### 3.1 What Goes Wrong

The central obstacle: G_Wheeler ≠ G_fakeon at loop level.

The Wheeler propagator in momentum space is G_W(p) = P.V. 1/(p² − m²). In a loop integral, this Cauchy principal value introduces both +iε and −iε poles simultaneously. Anselmi (2020) showed this leads to three failures:

1. **Renormalizability violation:** The coexistence of ±iε prescriptions in loop integrals produces UV divergences that cannot be absorbed by the standard counterterms of the quadratic gravity action.
2. **Unitarity violation:** The optical theorem (which equates the imaginary part of forward scattering amplitudes to total cross sections) is violated because the principal value prescription does not correctly separate absorptive from dispersive parts.
3. **Stability violation:** The Hamiltonian constructed from Wheeler-quantized fields is unbounded below — the classical stability of the Wheeler propagator does not survive quantization at the loop level.

### 3.2 The Fakeon's Loop-Level Procedure

Anselmi's correct procedure (JHEP 11(2021)030, "Diagrammar of physical and fake particles") is:

1. Write all Feynman diagrams using the standard Feynman iε prescription for ALL propagators (including the ghost).
2. Apply spectral optical identities to decompose each diagram into threshold contributions — each threshold corresponds to cutting the diagram and placing a set of internal lines on-shell.
3. **Identify and drop every threshold that places a fakeon line on-shell.** This is the "fakeon projection."
4. The remaining amplitude is unitary (satisfies the optical theorem for physical particles only) and renormalizable.

The procedure works at every loop order and for arbitrary diagrams (proven for bubble, triangle, box, pentagon, hexagon at one loop; triangle-with-diagonal, box-with-diagonal at two loops; arbitrary topology by induction).

### 3.3 Can Causal Order Reproduce This?

**Honest assessment: Not currently, but a path exists.**

The gap is: the SJ construction is defined for free fields on fixed backgrounds. It has no loop-level extension. To reproduce the fakeon's loop-level procedure from causal order, one would need:

1. **An interacting SJ construction on causal sets.** Free-field SJ is well-defined (Johnston 2009). Interacting SJ on causal sets is active research — Dowker and collaborators have done preliminary work on in-in correlators and scattering amplitudes on causal sets (arXiv:2402.08555, 2024). But the results are far from loop-level completeness.

2. **A causal-set version of the spectral optical theorem.** The spectral decomposition of amplitudes into thresholds has a natural causal interpretation: a threshold corresponds to a set of particles going on-shell simultaneously, which is a real physical process localized on a set of causally related events. Dropping fakeon thresholds corresponds to the causal order forbidding certain sets of on-shell events. But this interpretation hasn't been formalized.

3. **Extension to spin-2 fields on causal sets.** The SJ construction and the Belenchia et al. Wheeler-propagator result are for scalar fields only. Spin-2 fields on causal sets are essentially unexplored. The vierbein formalism on causal sets (Dowker, Surya) might provide a path, but no construction exists.

### 3.4 CFT's Position: A Well-Posed Open Problem

CFT does not claim to solve the loop-level problem. It states the problem precisely:

**Open Problem (Loop Extension):** Develop the Sorkin-Johnston vacuum construction for interacting fields on causal sets and show that:
(a) The free-field SJ construction for negative-spectral-weight modes produces the Wheeler propagator (partially proven by Belenchia et al. 2015 for scalar unstable modes).
(b) The interacting extension, when applied to loop diagrams involving negative-spectral-weight modes, reproduces Anselmi's threshold-dropping procedure.

If (a) and (b) are proven, CFT would provide a complete causal-order derivation of the fakeon prescription at all loop orders. If (b) fails, CFT remains valid as a tree-level motivation but cannot claim to derive the full fakeon prescription.

**Why this problem is tractable (if difficult):** The threshold-dropping procedure has a clear physical meaning — prevent ghosts from going on-shell. The causal-order interpretation is equally clear — the causal structure forbids modes with negative spectral weight from materializing. The challenge is purely technical: connecting the causal-set formalism (discrete, combinatorial) to the diagrammatic formalism (continuum, analytic) at loop level.

### What CFT Explains That QG+F Doesn't

1. **Why the fakeon:** QG+F says "the ghost must be removed for unitarity." CFT says "the causal order of spacetime cannot support a mode with negative spectral weight as a real particle — the SJ vacuum construction breaks down, and the only ≺-compatible propagator is the Wheeler propagator." This is a constructive derivation, not just a consistency requirement.

2. **Why microcausality is violated:** QG+F treats it as a cost. CFT identifies it as the continuum signature of the discrete causal substructure.

3. **Why analyticity is sacrificed:** QG+F has no explanation. CFT identifies it as the S-matrix encoding of causally forbidden intermediate states.

4. **Why d_s = 2 appears universally:** QG+F notes it as a propagator property. CFT explains it as the universal signature of causal set dynamics at short distances.

### What CFT Does NOT Explain Better Than QG+F

1. Loop-level unitarity proofs — QG+F has them; CFT conjectures them.
2. Renormalization group flow — identical, since the action is the same.
3. Inflationary predictions — identical (same action, same perturbation theory).
4. Non-perturbative physics — neither framework has results.

## 5. Novel Predictions

### 5.1 Prediction: Dimensional Crossover at the Fakeon Mass Scale

**CFT predicts that the spectral dimension crossover d_s = 4 → 2 occurs specifically at the fakeon mass M₂**, with the transition governed by the causal set's nonlocal d'Alembertian. Standard QG+F predicts d_s = 2 in the UV but does not link the crossover scale to any specific physical mechanism.

Concretely: the return probability for a diffusion process on the quantum spacetime satisfies:

    P(σ) ~ σ^{−d_s/2}

with d_s = 4 for diffusion time σ ≫ 1/M₂² and d_s = 2 for σ ≪ 1/M₂². The crossover profile is determined by the specific nonlocality of the causal set d'Alembertian and may differ from the naive QG+F prediction (smooth interpolation from the propagator). If the profiles differ, this is a distinguishing prediction — in principle testable via cosmological observables sensitive to short-distance geometry (CMB non-Gaussianities, primordial gravitational wave spectrum modifications).

**Testability:** Currently speculative. The prediction becomes concrete only if the crossover profile can be computed from both the QG+F propagator and the causal set d'Alembertian and shown to differ.

### 5.2 Prediction: Microcausality Violation Window = Discreteness Scale

**CFT identifies the microcausality violation scale 1/M₂ with the causal set's fundamental discreteness scale ℓ.** This is an identification, not an independent prediction — but it has a consequence: the structure of the violation should bear signatures of discreteness.

In standard QG+F, the microcausality violation is smooth (it comes from the principal-value prescription in the continuum). In CFT, the violation arises from the coarse-graining of the discrete causal order. At scales just above ℓ, the violation should exhibit **fluctuations** characteristic of Poisson-sprinkled causal sets — not a smooth envelope but a stochastic profile with variance determined by the sprinkling density ρ ~ M₂⁴.

This is in principle distinguishable from the smooth QG+F prediction, though any measurement would require probing Planck-scale physics.

### 5.3 Structural Prediction: Causal Set Continuum Limit Should Produce Quadratic Gravity

**CFT implies that the continuum limit of causal set dynamics should produce the quadratic curvature action, not just the Einstein-Hilbert action.** Currently, the Benincasa-Dowker action recovers S_EH. CFT requires that higher-order causal set operators (involving longer chains of causal relations) produce the R² and C² terms. This is a falsifiable structural prediction for the causal set program itself: if no causal set action produces quadratic curvature terms in the continuum limit, CFT's foundation is weakened.

## 6. Assessment and Open Problems

### What CFT Achieves

1. **Conceptual unification:** Connects two independent research programs (causal sets and QG+F) through the concrete mathematical identity G_fakeon = G_Wheeler at tree level.
2. **Explanatory depth:** Provides causal-order explanations for four of QG+F's explanatory debts: fakeon origin, microcausality meaning, analyticity sacrifice, and d_s = 2 universality.
3. **Spacetime microstructure:** Unlike bare QG+F, CFT has a microscopic picture — the causal set — that gives physical meaning to the continuum theory's formal structures.

### What CFT Does Not Achieve

1. **Loop-level derivation:** The central conjecture (loop extension) is unproven. Without it, CFT is a tree-level motivation, not a complete derivation.
2. **Spin-2 SJ construction:** The causal-order argument is rigorously established only for scalar fields. The extension to spin-2 (which is the actual ghost) is unexplored.
3. **Non-perturbative completion:** CFT offers the causal set as a candidate discretization but has no non-perturbative results.
4. **Quantitative predictions:** The novel predictions (Section 5) are qualitative or structural, not quantitative. No numerical value is predicted that differs from QG+F.

### Circularity Concern

The most serious conceptual problem: CFT uses the causal order ≺ to constrain the propagation of the spin-2 fakeon χ_μν, but χ_μν is a perturbation of the metric that defines ≺. Within perturbation theory this is self-consistent (Axiom 4): the background ≺₀ determines prescriptions, and the fakeon's purely virtual nature ensures it doesn't invalidate ≺₀. Beyond perturbation theory, the circularity is unresolved. This is shared with any perturbative QG approach and is not specific to CFT.

### Key Open Problems

1. **Loop Extension Conjecture:** Develop the interacting SJ construction on causal sets and verify whether it reproduces threshold-dropping at loop level.
2. **Spin-2 SJ Construction:** Extend the SJ vacuum construction from scalar to tensor fields on causal sets.
3. **Quadratic Gravity from Causal Sets:** Find a causal set action whose continuum limit includes the R² + C² terms of quadratic gravity.
4. **Crossover Profile:** Compute the spectral dimension crossover profile from both QG+F and causal set dynamics and determine whether they differ.
5. **Non-perturbative Self-Consistency:** Establish whether the causal set discretization provides a non-perturbative completion of QG+F (analogous to lattice QCD for QCD).

### Overall Verdict

CFT is a **well-motivated tree-level framework** that provides genuine explanatory value over bare QG+F — particularly for the fakeon interpretation (QG+F's #1 explanatory debt) and microcausality violation. Its central weakness is the unproven loop-level conjecture, which is the theory's make-or-break challenge. The theory does not yet produce quantitative predictions that differ from QG+F, but it identifies concrete structural predictions (Section 5.3) that could be tested within the causal set program.

CFT is best understood as a **research program** — a specific hypothesis about the physical origin of the fakeon that generates well-posed mathematical questions (Open Problems 1-5) and connects two otherwise disjoint communities (causal set theory and QG+F). Its value lies not in competing with QG+F on calculations but in providing the conceptual foundation that QG+F lacks.
