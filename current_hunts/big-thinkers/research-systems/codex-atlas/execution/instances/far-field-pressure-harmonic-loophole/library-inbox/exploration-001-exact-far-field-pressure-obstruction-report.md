# Exploration 001: Reconstruct the Exact Far-Field Pressure Obstruction

## 1. Executive Verdict

- [VERIFIED] In Vasseur's actual De Giorgi recurrence, the term that forces `beta < 4/3` is **not** the harmonic nonlocal pressure `P_{k1}`. It is the **local non-divergence pressure term** `P_{k21}` appearing in Step 5 of Proposition 3.
- [VERIFIED] The harmonic / far-field piece already appears in Vasseur as `P_{k1}`, harmonic on the interior ball, and for the relevant regime `p > 10` it is controlled with exponent `> 3/2`; it is not the load-bearing obstruction.
- [CHECKED] The mission's surviving "far-field harmonic loophole" therefore does **not** coincide with the literal bottleneck term in the Vasseur split. It is a proposed **alternative pressure decomposition** idea: replace the bad pressure interaction by a near-field / far-field split in which the far-field piece is harmonic on `Q_k` and might pair more favorably with the De Giorgi test function.
- [CHECKED] The exact obstruction to carry into the Tao filter is thus a **decomposition mismatch**: can a harmonic far-field split actually recapture the dangerous pressure contribution that Vasseur isolates as local `P_{k21}`, and if so, does harmonicity make that coefficient `U_k`-dependent rather than fixed at energy scale?

## 2. Exact Formulas and Pressure Decomposition

### 2.1 De Giorgi setup

From Vasseur (2007), Section 4:

- `v_k = [|u| - (1 - 2^{-k})]_+`
- `Q_k = [T_k,1] x B_k`
- `U_k = sup_{t in [T_k,1]} ∫_{B_k} |v_k|^2 dx + ∫∫_{Q_k} |d_k|^2 dx dt`

Lemma 11 / equation (11) gives the truncated energy inequality

```text
∂_t(v_k^2/2) + div(u v_k^2/2) + d_k^2 - Δ(v_k^2/2)
+ div(uP) + (v_k/|u| - 1) u · ∇P ≤ 0.
```

After multiplying by the cutoff `η_k` and integrating, equation (13) yields

```text
U_k ≤ C 2^{6k} ∫∫_{Q_{k-1}} |v_k|^2
    + C 2^{3k} ∫∫_{Q_{k-1}} |v_k|^3
    + 2 ∫_{T_{k-1}}^1 |∫ η_k {div(uP) + (v_k/|u| - 1) u · ∇P} dx| dt.
```

The pressure contribution in the last line is the only place where the full recurrence can still fail.

### 2.2 Vasseur's pressure split

On the interior region one decomposes

```text
P|_{B_{k-2/3}} = P_{k1} + P_{k2},
```

where:

- [VERIFIED] `P_{k1}` is the **nonlocal / harmonic** part on the local ball.
- [VERIFIED] `P_{k2}` is the local part solving

```text
-ΔP_{k2} = Σ_{i,j} ∂_i∂_j (φ_k u_i u_j).
```

Then Vasseur splits the local pressure further:

```text
P_{k2} = P_{k21} + P_{k22} + P_{k23},
```

with

```text
-ΔP_{k21}
= Σ_{i,j} ∂_i∂_j [ φ_k u_j (1 - v_k/|u|) u_i (1 - v_k/|u|) ],
```

and `P_{k22}, P_{k23}` the mixed and fully truncated pieces.

### 2.3 What the literal far-field object is

- [VERIFIED] In the Vasseur split, the harmonic far-field object is `P_{k1}`.
- [VERIFIED] In the surviving loophole language from the H^1 dead-end note, the candidate object is instead a **different** local/far split, usually of the form

```text
p = p_near + p_far,  with  Δp_far = 0 on Q_k,
```

where `p_far` is generated by sources outside the local cylinder/ball and is therefore harmonic inside it.

- [CHECKED] This candidate split is conceptually closer to a Wolf-type harmonic + particular decomposition than to Vasseur's `P_{k1} / P_{k21}` split.

### 2.4 Immediate reconciliation

- [VERIFIED] Vasseur's harmonic term `P_{k1}` is already favorable.
- [CHECKED] Therefore the loophole can only matter if an alternative far/near decomposition absorbs part of what Vasseur calls the bad local term `P_{k21}` into a harmonic far-field contribution.

That is the exact structural question the next exploration must test.

## 3. Recurrence-Slot Reconstruction

### 3.1 The harmless harmonic term

For the nonlocal harmonic pressure `P_{k1}`, Vasseur's Step 4 gives a bound with exponent

```text
β_nonlocal = (5/3)(1 - 1/p),
```

which is `> 3/2` when `p > 10`. In the regime relevant to the obstruction, this term is not the bottleneck.

### 3.2 The actual bad slot

The dangerous term is Step 5b for `P_{k21}`. Using equation (19), the pressure interaction splits into a divergence piece and a non-divergence piece:

```text
div(u P_{k21}) + (v_k/|u| - 1) u · ∇P_{k21}
= div( ... ) - P_{k21} div(u v_k/|u|).
```

The divergence part is favorable. The non-divergence part is the obstruction:

```text
I_k
= ∫∫_{Q_{k-1}} |P_{k21}| |div(u v_k/|u|)| dx dt
≤ ||P_{k21}||_{L^q}
   ||d_k||_{L^2}
   ||1_{v_k>0}||_{L^{2q/(q-2)}}.
```

Now:

- [VERIFIED] `||P_{k21}||_{L^q} ≤ C_q`
- [VERIFIED] `||d_k||_{L^2} ≤ U_{k-1}^{1/2}`
- [VERIFIED] `||1_{v_k>0}||_{L^{2q/(q-2)}} ≤ C^k U_{k-1}^{5(q-2)/(6q)}`

Hence

```text
I_k ≤ C_q C^k U_{k-1}^{ 1/2 + 5(q-2)/(6q) }
    = C_q C^k U_{k-1}^{ 4/3 - 5/(3q) }.
```

Letting `q -> ∞` gives the load-bearing barrier

```text
β_bad = 4/3.
```

### 3.3 The `1/2 + 5/6` arithmetic

- [VERIFIED] The `1/2` comes from the gradient factor `||d_k||_{L^2}`.
- [VERIFIED] The `5/6` comes from the De Giorgi level-set measure chain

```text
H^1 -> L^6
=> L^{10/3}_{t,x}
=> |{v_k>0}| ≤ C^k U_{k-1}^{5/3}
=> ||1_{v_k>0}||_{L^2} ≤ C^k U_{k-1}^{5/6}.
```

So the exact recurrence slot is:

```text
local non-divergence pressure
= fixed pressure size × gradient factor × level-set indicator factor
= U_{k-1}^0 × U_{k-1}^{1/2} × U_{k-1}^{5/6}
= U_{k-1}^{4/3}.
```

## 4. Why the Coefficient Is Fixed Rather Than `U_k`-Dependent

### 4.1 In the Vasseur split

- [VERIFIED] The source of `P_{k21}` contains the factors `u (1 - v_k/|u|)`, which are bounded independently of the shrinking level sets.
- [VERIFIED] As a result, Calderon-Zygmund only gives a **constant-scale** bound

```text
||P_{k21}||_{L^q} ≤ C_q,
```

not a bound with any positive power of `U_{k-1}`.

This is the exact sense in which the coefficient is fixed instead of `U_k`-dependent.

### 4.2 In the H^1 / far-field framing

The H^1 dead-end note phrases the remaining lead differently:

- local pressure closes,
- far-field pressure remains,
- `||p||_{H^1} ≤ C E_0`.

So in that framing the pressure size is again controlled at **energy scale**, not at De Giorgi scale. Harmonicity might improve oscillation or local regularity, but unless it changes the actual pairing so that a positive power of `U_{k-1}` appears, the coefficient stays fixed `O(E_0)`.

### 4.3 Important notation warning

- [CHECKED] The H^1 dead-end note says "far-field pressure is the sole obstruction" and also refers to `P_k^{21}`. In literal Vasseur notation this is inconsistent, because `P_k^{21}` is the bad **local** piece, while the harmonic nonlocal piece is `P_{k1}`.
- [CHECKED] The most coherent reading is: the note was speaking conceptually about an alternative local/far split, not literally about Vasseur's notation.

This is exactly why the next exploration has to test Tao-compatibility at the level of the **alternative harmonic far-field decomposition**, not merely repeat the Vasseur split.

## 5. Compact Tao-Filter Statement

The next exploration only needs the following equation-level question.

Assume one rewrites the local pressure on `Q_k` as

```text
p = p_near + p_far,
```

with

```text
Δp_far = 0 on Q_k
```

and with the claimed gain coming from estimating a harmonic pairing such as

```text
∫∫_{Q_k} p_far ψ_k dx dt
```

or its equivalent De Giorgi test-function form.

Then the loophole survives only if **all** of the following hold for averaged Navier-Stokes:

1. [CHECKED] the averaged pressure still admits the same local near/far decomposition with `p_far` harmonic on the working cylinder;
2. [CHECKED] the Harnack / oscillation mechanism for `p_far` survives averaging;
3. [CHECKED] that harmonic estimate changes the pressure contribution from a fixed energy-scale coefficient to a genuinely `U_k`-dependent one.

If averaging preserves harmonic far-field structure but the coefficient remains only `O(E_0)` or `O(1)`, then the loophole is Tao-compatible and closes.
