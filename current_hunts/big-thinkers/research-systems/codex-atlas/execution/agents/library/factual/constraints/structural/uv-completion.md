# UV Completion / Renormalizability (Constraint A5)

**Statement:** The theory must make sense at arbitrarily high energies (or arbitrarily short distances). This can be achieved through:
- (a) Perturbative renormalizability (all divergences absorbed by finite number of counterterms)
- (b) Asymptotic safety (non-perturbative UV fixed point)
- (c) UV finiteness (no divergences at all, as in string theory)
- (d) Fundamental discreteness (no trans-Planckian physics, as in LQG/causal sets)

**Mathematical form:**
- Perturbative: Power-counting requires [G_N] = -2 in mass units, making gravity perturbatively non-renormalizable in D=4
- Asymptotic safety: lim_{k→∞} G_k = G* (fixed point) with finite-dimensional critical surface
- String theory: Amplitudes A_n ~ ∫ d(moduli) × (worldsheet correlators) are UV finite order by order

**Restrictiveness: MAXIMALLY RESTRICTIVE**

This is the fundamental obstacle in quantum gravity. Pure GR quantized perturbatively fails — the 2-loop divergence in pure gravity (Goroff-Sagnotti 1986) and 1-loop divergences in gravity + matter are non-renormalizable. This single constraint eliminates the most naive approach and forces one of the above escape routes.

## The Four Known Escape Routes

Each escape route defines a major research program:

1. **String theory:** UV-finite amplitudes from extended objects (strings) replacing point particles. No higher-derivative ghosts because the propagator is modified by the entire string spectrum.
2. **Asymptotic safety:** Non-perturbative fixed point makes Newton's constant run to a finite value G* in the UV. Finite-dimensional critical surface preserves predictivity.
3. **LQG / Causal Sets:** Fundamental discreteness at the Planck scale eliminates trans-Planckian physics entirely. No continuum divergences because there is no continuum at small scales.
4. **Infinite Derivative Gravity (IDG):** Entire functions in the propagator (e.g., exp(-□/M²)) achieve UV improvement without introducing new poles (ghosts).

## Why This Is the Most Fundamental Constraint

UV completion, combined with unitarity + ghost freedom, creates the single tightest bottleneck in quantum gravity. Adding R² terms makes gravity renormalizable (Stelle 1977) but introduces ghosts → unitarity violation. Each escape route resolves this tension differently. Any novel approach must identify its own resolution.

## Dependency

UV completion is genuinely independent — cannot be derived from other constraints.
