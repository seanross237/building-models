# Causal Fakeon Theory (CFT): Fakeon from Spacetime's Causal Order

## Summary

Causal Fakeon Theory (CFT) is a formal framework connecting the causal partial order of spacetime to the fakeon prescription in quadratic gravity. The massive spin-2 ghost cannot become a real particle because doing so would violate the fundamental causal structure of spacetime. The framework rests on a real, unexploited mathematical connection: both causal set theory and the fakeon formalism independently produce the Wheeler propagator for problematic degrees of freedom.

**Status:** Well-motivated tree-level framework with genuine explanatory value over bare QG+F. Central challenge is the unproven loop-level conjecture. Best understood as a research program generating well-posed mathematical problems connecting two disjoint communities (causal sets and QG+F).

**Self-assessment:** Novelty 7/10, Consistency 5/10, Clarity 8/10, Viability 6/10.

## The Wheeler Propagator Connection

The fakeon propagator is the Cauchy principal value of the pole:

    G_fakeon(p) = P.V. 1/(p² - m²)

In position space, this equals (G_R + G_A)/2 — the Wheeler propagator (half-advanced + half-retarded), and the real part of the Feynman propagator. Anselmi is aware of this identity (JHEP 03(2020)142).

Independently, **Belenchia, Benincasa & Liberati (JHEP 03(2015)036, arXiv:1411.6513)** studied nonlocal scalar QFT from causal set dynamics and found: in 4D, the unstable modes of the causal set d'Alembertian are propagated via the Wheeler propagator and hence do not appear in asymptotic states — effectively purely virtual.

**The gap nobody has exploited:** Causal set structure independently produces fakeon-like behavior for problematic DOF, through the same mathematical object. The causal set community doesn't discuss fakeons; Anselmi doesn't discuss causal sets.

## Formal Axiom Structure

CFT rests on four axioms (ordered from most established to most speculative) and one conjecture.

### Axiom 1 — Causal Foundation
Physical spacetime is fundamentally a locally finite partial order (causal set) C = (𝒞, ≺). The partial order determines conformal geometry (Malament's theorem, 1977); discrete volume measure (element counting) determines the conformal factor (Sorkin's "order + number = geometry"). **Status:** Established mathematics. The Benincasa-Dowker action recovers the Einstein-Hilbert action in the continuum limit.

### Axiom 2 — Spectral Causal Compatibility
The vacuum state and propagators of every quantum field on C are determined by the causal structure via the Sorkin-Johnston (SJ) construction (Johnston 2009, Afshordi-Aslanbeigi-Sorkin 2012). The commutator function Δ(x,y) = [ϕ(x), ϕ(y)] is computed from the retarded Green function (depends only on causal order). The SJ vacuum is defined by the positive part of the spectrum of iΔ.

**The causal compatibility condition:**
- **Positive spectral weight (r > 0):** SJ construction yields well-defined Fock vacuum → Feynman propagator G_F → on-shell particle production permitted.
- **Negative spectral weight (r < 0):** SJ construction yields negative-norm states incompatible with probabilistic interpretation → causal order **cannot support this mode as a physical particle**.

**Status:** SJ construction rigorously defined for free scalar fields on globally hyperbolic spacetimes and on causal sets. The extension to negative-spectral-weight modes is CFT's novel element — the claim that the SJ construction's breakdown for ghost modes is the physical mechanism enforcing the fakeon prescription.

### Axiom 3 — Wheeler Exclusion
Modes excluded from the physical spectrum by Axiom 2 propagate via the Wheeler propagator G_W = (G_R + G_A)/2, the unique ≺-compatible treatment for non-physical modes. This follows from: (1) G_W has no on-shell poles — no particle production; (2) Belenchia et al. (2015) proved causal set dynamics produce precisely G_W for unstable modes in 4D; (3) G_W is time-reversal symmetric — for a mode with negative residue and anti-causal energy flow (Donoghue 2019), the only ≺-compatible propagator has no preferred temporal direction.

### Axiom 4 — Perturbative Self-Consistency
The causal order ≺ is dynamically generated by the theory itself — the metric g_μν (defining ≺ in the continuum limit) solves the quadratic gravity field equations. The fakeon χ_μν is a perturbation of ≺. Self-consistency requires that the fakeon contributes to the full causal order only through virtual (loop) effects, never as an on-shell disturbance that could modify ≺₀. This is self-consistent precisely because the fakeon is purely virtual: it corrects the background geometry at the quantum level (renormalization of Newton's constant, running couplings) but never produces a classical on-shell perturbation that would invalidate the background causal structure used to define it.

**Status:** Valid within perturbation theory (where it functions as a consistency check). Beyond perturbation theory, circularity is unresolved — shared with any perturbative QG approach.

### Conjecture — Loop Extension (Central Open Problem)
The tree-level Wheeler prescription extends to Anselmi's full loop-level fakeon prescription at every order. At loop level, G_Wheeler ≠ G_fakeon: naive use of Wheeler propagator in loops violates renormalizability, unitarity, and stability (Anselmi 2020, due to coexistence of ±iε). Anselmi's correct procedure (JHEP 11(2021)030): (1) compute loops with standard Feynman iε for all propagators; (2) apply spectral optical theorem to decompose into threshold contributions; (3) drop all thresholds involving fakeon frequencies; (4) remaining amplitude satisfies unitarity. Proven for arbitrary diagram topology by induction.

**CFT's conjecture:** Dropping fakeon thresholds corresponds to the causal order forbidding the ghost from materializing at any loop order. The SJ construction, extended to interacting fields on causal sets, should reproduce this threshold-dropping procedure.

## Physical Consequences

### Recovery of QG+F at Tree Level
CFT reproduces standard QG+F at tree level by construction. The massless graviton (positive residue) receives Feynman prescription via Axiom 2. The massive spin-2 (negative residue) receives Wheeler/fakeon prescription via Axiom 3. The massive scalar (positive residue) receives Feynman prescription. Tree-level scattering amplitudes, Newton's law, Stelle potential, and all GR tests are identical to QG+F.

### Microcausality Violation as Causal Granularity
In QG+F, microcausality violation below ~1/M₂ is treated as an acceptable cost (Anselmi 2026 goes further: causality should be "abandoned in fundamental physics"). CFT reframes this as a **signature of discrete causal order**: below the discreteness scale ℓ ~ 1/M₂, the partial order ≺ is not well-defined — events are neither causally related nor spacelike-separated but simply *unresolved*. The violation is a prediction, not a sacrifice.

Furthermore, in standard QG+F the violation is smooth (continuum principal-value prescription). In CFT, the violation arises from coarse-graining of the discrete causal order, and should exhibit **stochastic fluctuations** characteristic of Poisson-sprinkled causal sets, with variance ρ ~ M₂⁴. In principle distinguishable from smooth QG+F prediction, though probing requires Planck-scale physics.

### Spectral Dimension Flow d_s = 4 → 2 Unified
QG+F gives d_s = 2 in the UV from 1/p⁴ propagator. Independently, Belenchia et al. (Phys. Rev. D 93, 044017, 2016) showed causal set d'Alembertians exhibit **universal dimensional reduction to d_s = 2 in all spacetime dimensions**. CFT unifies these: the 1/p⁴ propagator is the continuum expression of the nonlocal causal set d'Alembertian. The d_s = 2 reduction has a common causal origin. CFT predicts the crossover scale is M₂: d_s = 4 for E ≪ M₂, flowing to d_s = 2 for E ≫ M₂.

### Analyticity Sacrifice Explained
Analyticity of the S-matrix requires all intermediate states in scattering be physical (Cutkosky rules connect imaginary parts to on-shell states). The fakeon removes certain intermediate states from the physical spectrum. The analyticity failure encodes the fact that the causal order forbids certain modes from materializing — "missing" analytic structure = "missing" on-shell fakeon states. This is a conceptual explanation, not a derivation, but transforms the sacrifice from an unmotivated cost to a consequence of causal-order enforcement.

### What CFT Explains That QG+F Doesn't
1. **Why the fakeon** — constructive derivation from SJ breakdown, not just consistency requirement
2. **Why microcausality is violated** — continuum signature of discrete causal substructure
3. **Why analyticity is sacrificed** — S-matrix encoding of causally forbidden intermediate states
4. **Why d_s = 2 universally** — universal signature of causal set dynamics at short distances

### What CFT Does NOT Explain Better Than QG+F
1. Loop-level unitarity proofs — QG+F has them; CFT conjectures them
2. Renormalization group flow — identical (same action)
3. Inflationary predictions — identical (same perturbation theory)
4. Non-perturbative physics — neither framework has results

## Novel Predictions

1. **Dimensional crossover at M₂:** Spectral dimension transition governed by causal set nonlocal d'Alembertian; crossover profile (return probability P(σ) ~ σ^{−d_s/2}) may differ from naive QG+F smooth interpolation — in principle testable via CMB non-Gaussianities or primordial GW spectrum modifications. Currently speculative; becomes concrete only if profiles can be computed from both frameworks and shown to differ.

2. **Microcausality violation window = discreteness scale:** Identifies 1/M₂ with causal set fundamental discreteness ℓ; violation should exhibit stochastic fluctuations (not smooth envelope), with variance from sprinkling density ρ ~ M₂⁴. In principle distinguishable from smooth QG+F, but requires Planck-scale probing.

3. **Structural prediction (falsifiable within CST):** The continuum limit of causal set dynamics should produce the quadratic curvature action (R² + C²), not just Einstein-Hilbert. Currently, the Benincasa-Dowker action recovers only S_EH. CFT requires higher-order causal set operators (involving longer chains of causal relations) to produce the quadratic terms. If no causal set action produces quadratic curvature in the continuum limit, CFT's foundation is weakened.

## The Loop-Level Problem (Central Obstacle)

### What Goes Wrong
At loop level, naive use of Wheeler propagator introduces ±iε poles simultaneously, causing: (1) renormalizability violation — UV divergences not absorbable by standard counterterms; (2) unitarity violation — optical theorem fails; (3) stability violation — Hamiltonian unbounded below (Anselmi, JHEP 03(2020)142).

### Three Technical Requirements to Bridge the Gap
1. **Interacting SJ construction on causal sets** — Free-field SJ well-defined (Johnston 2009). Interacting SJ on causal sets is active research: Dowker et al. (arXiv:2402.08555, 2024) have preliminary in-in correlator and scattering amplitude results, far from loop-level completeness.
2. **Causal-set spectral optical theorem** — Threshold decomposition needs causal interpretation: thresholds correspond to on-shell events; dropping fakeon thresholds = causal order forbidding certain on-shell events. Not yet formalized.
3. **Spin-2 SJ construction** — SJ construction and Belenchia et al. Wheeler result are for scalar fields only. Spin-2 fields on causal sets essentially unexplored; vierbein formalism (Dowker, Surya) might provide a path.

### Tractability Assessment
The problem is well-posed: threshold-dropping has clear physical meaning (prevent ghosts from going on-shell), and the causal-order interpretation is equally clear (causal structure forbids negative-spectral-weight modes from materializing). The challenge is purely technical: connecting the discrete/combinatorial causal-set formalism to the continuum/analytic diagrammatic formalism at loop level.

## Key Open Problems

1. **Loop Extension:** Develop interacting SJ construction on causal sets; verify it reproduces Anselmi's threshold-dropping at loop level.
2. **Spin-2 SJ Construction:** Extend SJ vacuum construction from scalar to tensor fields on causal sets.
3. **Quadratic Gravity from Causal Sets:** Find a causal set action whose continuum limit includes R² + C² terms.
4. **Crossover Profile:** Compute spectral dimension crossover from both QG+F propagator and causal set d'Alembertian; determine if they differ.
5. **Non-perturbative Self-Consistency:** Establish whether causal set discretization provides non-perturbative completion of QG+F (analogous to lattice QCD).

## Devil's Advocate: Key Objections

1. **Loop-level conjecture unproven (MOST SERIOUS):** Without it, CFT is tree-level motivation, not a complete derivation. QG+F has proven loop-level unitarity; CFT only conjectures it.
2. **Spin-2 gap:** The causal-order argument is rigorously established only for scalar fields. The actual ghost is spin-2.
3. **Possibly just unitarity in causal language:** The added content beyond the standard unitarity argument may be thin; main value is constructive derivation rather than mere requirement.
4. **Cannot select fakeon independently:** Requires knowing the theory has negative-residue poles first.
5. **Causal sets don't produce quadratic gravity:** Benincasa-Dowker action gives Einstein-Hilbert, not R² + C². Connection is at mode-treatment level only.
6. **No quantitative predictions:** Novel predictions (Section above) are qualitative or structural, not quantitative. No numerical value differs from QG+F.
