# Quantum Field Theory on Causal Sets: The Sorkin-Johnston Formalism

## Overview

The Sorkin-Johnston (SJ) formalism provides a framework for constructing quantum field theories on causal set backgrounds. Unlike standard QFT which relies on the continuum structure of spacetime, this approach uses only the causal structure and the discrete d'Alembertian to define matter field theories.

## How It Works

The SJ formalism starts with the advanced and retarded Green's functions on the causal set and the spacetime commutator (Peierls bracket) to construct a unique Wightman two-point function for a free Gaussian scalar field. This approach is applicable to both continuum spacetime regions and discrete causal set structures.

A key feature is that Green functions and unique two-point correlation functions can be obtained purely from the causal set data, without reference to any background continuum.

## Nonlocal QFT from Causal Sets

Belenchia, Benincasa, and Liberati (2015) derived a nonlocal scalar quantum field theory in flat spacetime from the dynamics of a scalar field on a causal set. Their key finding: "this non-local QFT contains a continuum of massive modes in any dimension."

In 2 dimensions, the Hamiltonian is positive definite, yielding a well-defined quantum theory. In 4 dimensions, complications arise from unstable modes, which are handled via "the so called Wheeler propagator" that prevents them from appearing in asymptotic states. Despite this, the non-positive-definite Hamiltonian in four dimensions leaves "potential issues with the quantum theory."

## Interacting Fields and Recent Advances

Recent work has extended the formalism beyond free fields to interacting scalar field theories on causal sets. Advances in 2024 include diagrammatic expansions for in-in correlators and scattering amplitudes, providing new calculational tools for understanding early Universe observables such as cosmological correlation functions.

## Fermionic Fields

The SJ formalism has also been extended to fermionic matter fields on causal set backgrounds, though this area is less developed than the scalar field case.
