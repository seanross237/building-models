# Sensitivity of BMEG Predictions to eta_h

If eta_h shifts from the standard AS value of +1.03 to some other positive value eta_h^{BMEG}, all four novel predictions of BMEG change quantitatively but remain qualitatively intact.

## Prediction-by-Prediction Analysis

### 1. Graviton Scattering Cross Section
- Formula: sigma ~ G_N^2 s^{eta_h - 1} ln(s)
- If eta_h < 1: cross section decreases more steeply with energy
- If eta_h > 1: increases slightly before logarithmic factor dominates
- Qualitative distinction from standard AS preserved for any eta_h != 0

### 2. Black Hole Evaporation
- The partial compensation of running G by the enhanced fluctuation propagator occurs for any eta_h > 0
- Complete evaporation (no remnants) requires eta_h > 0 but precise timescale depends on numerical value
- Qualitatively robust

### 3. Tensor Power Spectrum
- Blue tilt correction proportional to eta_h
- Smaller eta_h means smaller but still positive tilt
- Prediction remains qualitatively distinct from standard AS

### 4. Black Hole Entropy Log Correction
- Coefficient shifts with eta_h but the modified correction persists for any eta_h != 0
- The correction distinguishes BMEG from both standard AS and infinite derivative gravity (IDG)

## Summary

**All four predictions are qualitatively robust to changes in eta_h, as long as eta_h > 0.** Since the sign robustness arguments (antiscreening, spectral positivity, unitarity) are background-independent, BMEG's distinguishing predictions survive the self-consistency check.

The predictions shift quantitatively, but they remain qualitatively distinct from both standard AS (which treats background = fluctuation) and IDG (which has different UV structure).
