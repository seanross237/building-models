## Open-Eywa Operator Note

Open-Eywa is a system designed for executing complex, multi-step tasks by distributing work across a network of nodes. Each node represents a distinct unit of computation and state, making it the durable unit of work. This ensures that progress is maintained even if individual nodes encounter issues.

For a human operator inspecting this mission, the following files are recommended for initial review:

*   `input/goal.md`: This file defines the overall objective of the mission.
*   `plan.md`: This outlines the strategy and steps to achieve the mission's goal.
*   `state.md`: This provides the current status and any relevant context or intermediate results.

By examining these files, you can quickly understand the mission's purpose, the approach being taken, and the current progress.
