# Exploration 002 Goal

## Objective

Freeze exactly one admissibility decision and one primary comparator for
step-015, then judge whether the bridge burden or prior-art overlap already
forces a pre-package negative verdict.

## Success Criteria

- decide whether the live route is directly suitable-weak / LEI admissible or
  instead lives on one named proxy class;
- if proxy-based, write the exact bridge burden back to the theorem floor and
  say whether it looks cheaper than the target theorem, unresolved, or already
  theorem-scale;
- name one closest live geometric prior-art comparator only;
- state the exact shared structure already present there and the claimed delta
  of the proposed route; and
- identify the earliest honest kill condition if admissibility or comparator
  screening already fails before package choice.

## Failure Criteria

- leaves both direct and proxy admissibility live;
- names multiple comparators without choosing one primary comparator;
- reports only packaging differences rather than theorem-facing delta;
- or avoids stating whether the bridge burden is theorem-scale.
