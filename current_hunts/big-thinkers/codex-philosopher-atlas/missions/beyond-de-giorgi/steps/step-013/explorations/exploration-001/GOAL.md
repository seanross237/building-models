# Exploration 001 Goal

## Objective

Freeze the endpoint window for the active reformulation-audit chain:

- choose one primary endpoint family on the inherited LEI / CKN floor;
- choose one backup endpoint family only;
- freeze one intended solution class and one localization regime that later
  candidate reformulations must satisfy.

## Success Criteria

- exit with one primary endpoint family and one backup endpoint family, not a
  menu;
- explain why the primary endpoint is the correct first theorem-facing window
  for this branch;
- explain why the backup endpoint is the right reserve rather than a wider
  portfolio;
- state clearly which solution class and localization regime are frozen for
  later candidate testing.

## Failure Criteria

- endpoint choice remains a menu rather than a frozen pair;
- solution class or localization regime remains too vague to govern later
  localized equation work;
- the proposed endpoint pair drifts away from the inherited LEI / CKN floor.
