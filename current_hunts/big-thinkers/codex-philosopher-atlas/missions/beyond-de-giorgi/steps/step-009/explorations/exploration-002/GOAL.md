# Exploration 002 Goal

## Objective

Stress-test the cash-out, progress currency, and minimum threshold for the
best endpoint candidate in `step-009`.

The endpoint menu is already narrowed by the receptionist. This exploration
should test whether the likely freeze target, a sharpened estimate on the live
far-field pressure pairing inside the inherited local-energy contradiction
architecture, can support one non-cosmetic gain currency and one minimum
quantitative threshold that would still matter after later reconstruction and
any required localization debt.

## Required Questions

Answer all of the following from local repository materials:

1. If the endpoint is attached to the live far-field pairing, what exact
   positive deliverable would later tensor work have to produce:
   a smaller effective coefficient,
   cancellation of a named harmonic-moment family,
   a one-sided inequality on the full pairing,
   or another exact object?
2. Which progress currency is best supported:
   one-sided inequality,
   monotonicity,
   cancellation of a named harmonic-moment family,
   or quantified reduction of a specified nonlocal remainder?
3. What minimum quantitative threshold should `step-009` freeze now so later
   work cannot count cosmetic rephrasing as progress?
4. Why does that threshold still count as meaningful progress after later
   reconstruction and any localization debt, rather than merely rebranding
   coefficient-size control?
5. What source-backed negative tests rule out weaker thresholds, such as
   constant-mode quotienting, generic harmonic regularity, or improvements that
   do not shrink the actual bad coefficient?

## Working Hypotheses To Test, Not Assume

- The only clearly live cash-out line is an inequality of the form
  `|I_p^far| <= C_far,eff(k) 2^{12k/5} U_k^{6/5}`
  with `C_far,eff(k)` genuinely smaller than
  `||u||_{L^2}^2 / r_k^3`.
- The best progress currency is quantified reduction of the live nonlocal
  coefficient, with harmonic-moment cancellation admissible only if it is the
  mechanism for that reduction.
- The minimum meaningful threshold is a uniform positive-margin shrinkage of
  the live coefficient that survives later reconstruction and localization
  losses; "better modulo constants" is cosmetic.

Confirm, refine, or reject those hypotheses from source-backed evidence.

## Success Criteria

- Produce a ledger with columns:
  `candidate cash-out`, `currency`, `minimum threshold`,
  `why meaningful`, `failure mode of weaker version`.
- Label major claims `[VERIFIED]`, `[INFERRED]`, or `[PROPOSED]`.
- End with one recommended currency and one recommended minimum threshold only,
  or state plainly that no non-cosmetic threshold is currently supportable.

## Failure Criteria

- The report treats a representation identity as if it were itself a threshold.
- The report counts mode quotienting or generic harmonic smoothing as progress
  when the live coefficient is unchanged.
- The report sets a threshold that would be erased by the later reconstruction
  or localization bill.

If failure occurs, say so directly and recommend endpoint obstruction.
