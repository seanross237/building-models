# Meta Note - Step 009 Exploration 002

When an endpoint-first branch freezes a live bad term, the minimum threshold
must be stated as net improvement on that same term after the later debt is
repaid.

Reusable lesson:

- "better modulo constants" is not a threshold if constants are already killed;
- named cancellation only counts if it improves the same theorem-level
  inequality;
- and the right Step-1 threshold shape is often a uniform positive-margin net
  reduction in the frozen coefficient, not an unquantified improvement in
  spirit.

For this step, the admissible currency is shrinkage of the effective
far-field coefficient in the full `I_p^far` estimate. That prevents the branch
from surviving on surrogate gains, local-only improvements, or prettier tensor
packaging.
