# Meta Note - Step 009 Exploration 003

Admission rules should be frozen against the exact endpoint object, not against
branch rhetoric.

Reusable lesson:

- same-scale exterior or harmonic freedom is only a meaningful failure rule if
  it is phrased against the full theorem-level bad term;
- Tao sensitivity is only a screening rule unless it also changes the same
  theorem-level estimate;
- and a branch can survive an endpoint gate narrowly without admitting any
  tensor package yet.

For this step, both rules were frozen directly on the full `I_p^far` pairing.
That keeps later prior-art and packaging screening honest: no candidate may
survive by shifting to a surrogate term or by using Tao language as leverage.
