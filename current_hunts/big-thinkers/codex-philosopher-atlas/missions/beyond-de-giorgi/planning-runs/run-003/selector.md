# Selector - beyond-de-giorgi / run-003

## Full Ranking

1. **Chain 01 - Exact-Reformulation Bottleneck Audit**
   Best high-floor chain. It fixes one named bottleneck, forces estimate-level accountability, and still yields a strong obstruction memo if every reformulation collapses back to the same structure.

2. **Chain 04 - Critical Element / Concentration-Compactness With NS-Specific Rigidity**
   Best high-upside and most globally different chain. It is the clean alternative if local mechanism hunts stall, and even a negative result produces a reusable compactness-and-rigidity obstruction map.

3. **Chain 02 - Helical-Triad Differential Diagnosis**
   Best third slot for diversity. It tests a genuinely different NS-specific discriminator at the symbol and triad level, where Tao-style averaging may plausibly destroy exact structure without destroying generic harmonic analysis.

4. **Chain 03 - Pressure-Tensor Moment Route**
   Strong narrow survivor, but it overlaps more with Chain 01's exact-structure/local-mechanism hunt than Chain 02 does. In a three-chain portfolio, it is the first serious cut rather than a weak chain.

5. **Chain 05 - Geometry Reboot Under Open Package Change**
   Most condition-sensitive branch. It is worth reviving only if the explicit goal is to test whether the prior geometry kill was mainly a package artifact; otherwise too much of the budget goes into auditing setup costs.

## Why These Three Make The Strongest Portfolio

This selection gives the cleanest three-way spread across the run:

- **Chain 01** is the highest-floor local test. It asks the sharpest estimate-first question and is the most reliable producer of a useful obstruction memo if it dies.
- **Chain 02** is the best differentiated exact-structure probe. It moves from localized rewrites to helical and triadic interaction geometry, so it maximizes information gain if Chain 01 fails.
- **Chain 04** preserves the necessary high-upside gamble. It is the only route here that starts from a hypothetical blowup object and looks for an NS-specific rigidity mechanism.

Together they span:

- local estimate-level reformulation,
- spectral and algebraic interaction structure,
- global compactness and rigidity.

That is the least redundant portfolio available in this run while still preserving one high-ceiling path and maintaining useful failure outputs on all three branches.

## Why The Other Two Were Not Selected

- **Chain 03** narrowly misses because it is valuable but less portfolio-diversifying than Chain 02. If a fourth slot existed, it would be the next addition.
- **Chain 05** stays out because its value depends too heavily on whether the package-change hypothesis is itself the thing we want to test next.
