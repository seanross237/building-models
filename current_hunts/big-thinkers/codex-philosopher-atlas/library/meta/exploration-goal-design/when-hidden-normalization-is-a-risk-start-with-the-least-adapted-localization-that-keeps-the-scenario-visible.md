# When Hidden Normalization Is A Risk, Start With The Least Adapted Localization That Keeps The Scenario Visible

When a geometry branch is vulnerable to hidden normalization, fix the
localization package before the observable table and default to the least
adapted package that still keeps the target scenario visible.

That rule keeps the test honest. It prevents the exploration from inheriting a
preferred observable's geometry, and it forces any hoped-for coherence or
persistence to emerge from the data rather than from tube-fitted coordinates,
quantile thresholds, or other data-adaptive localization choices.

In Step 3 Exploration 002 this pointed to a neutral Eulerian parabolic package
for the filament/tube primary scenario, with explicit perturbation checks on
threshold and time window to expose whether the branch only survives by tuning
those choices.
