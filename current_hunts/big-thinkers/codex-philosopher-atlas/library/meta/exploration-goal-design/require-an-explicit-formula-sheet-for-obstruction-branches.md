# Require An Explicit Formula Sheet For Obstruction Branches

When a branch turns on a delicate obstruction, require the explorer to write the
exact live formula rather than paraphrasing the phenomenon.

In Exploration 001 this forced the report to identify the surviving
`I_p^far` pairing and its dominant term, which made the real obstruction easy to
file and compare later.
