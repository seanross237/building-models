# Write Forbid Novelty Claim Rules For Prior-Art Benchmarks

When a branch sits near a large prior-art literature, the exploration goal
should require an explicit "forbid novelty claim" rule for each candidate
family.

This makes prior-art overlap operational. It is more useful than a vague
similarity memo because it tells later steps exactly which formulations are
already disqualified as relabelings.
