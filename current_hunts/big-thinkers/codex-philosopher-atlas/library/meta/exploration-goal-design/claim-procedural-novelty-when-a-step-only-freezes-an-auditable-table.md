# Claim Procedural Novelty When A Step Only Freezes An Auditable Table

When a step has not found a new mechanism, the honest novelty claim is
procedural: it fixed the architecture, protocol, and bounded candidate table
that later screens can audit.

This is stronger than vague "setup work" language but weaker than pretending a
new gain has already been earned. It keeps the branch reusable without
overstating what the step proved.
