# Distinguish Native Action On The Frozen Burden From Post-Reconstruction Action

When a branch has already frozen one theorem-facing bad term, do not credit a
candidate with acting on that burden merely because the burden can be rewritten
after reconstructing other variables.

Ask a sharper question instead: does the candidate's native exact equation
already interface with the frozen burden, or does it only reach that burden
after an exchange step that reconstructs velocity, pressure, or other
nonlocal objects?

That gap is often the real obstruction. If the theorem-facing burden only
reappears after reconstruction, then the reconstruction, transfer, and
localization debts belong to the candidate's own ledger, and the branch should
not treat the surrogate-side equation as if it were already acting on the live
obstruction.
