Status: `VERIFIED` with `INFERRED` endpoint rule

For the frozen endpoint
`I_p^far = -\iint p_far div(v_k phi_k^2 e_hat)`, same-scale unconstrained
exterior or harmonic freedom is not a tolerable residual.

The branch-specific admission rule is exact:

- after full harmonic accounting and any required localization debt, a
  candidate is alive only if it returns to the complete `I_p^far` pairing with
  a genuinely smaller effective coefficient;
- constants do not matter because the pairing already kills them, but the live
  affine-and-higher harmonic moments still do;
- and a candidate that closes only on a local piece, self-induced piece,
  reduced model, or representation-level defect has not met the frozen
  threshold if the reconstructed full pairing still carries a same-scale
  unconstrained exterior or harmonic remainder.

So the kill condition is immediate: once the complete reconstructed pairing
cannot be converted into a positive-margin smaller coefficient, the branch
fails before that remainder can be presented as progress.
