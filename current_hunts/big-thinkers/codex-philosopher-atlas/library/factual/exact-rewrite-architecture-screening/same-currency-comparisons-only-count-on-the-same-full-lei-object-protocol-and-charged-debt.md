# Same-Currency Comparisons Only Count On The Same Full LEI Object Protocol And Charged Debt

Status: `INFERRED` same-ledger comparison rule

Direct same-currency comparison is valid only when two claims act on the same
frozen endpoint object:
the full localized `I_flux[phi]` bundle,
in the same suitable-weak / Leray-Hopf plus LEI solution class,
under the same shared CKN-style cutoff protocol,
and after the same pressure-recovery and admissibility debt has been charged.

Only in that setting may later steps compare net coefficient changes directly.
If the object, solution class, localization regime, or charged debt differs,
the branch has left the same ledger and no direct comparison is honest.
