# Success Requires A Smaller Effective LEI Cutoff-Flux Cost In The Same Protocol

Status: `INFERRED` success criterion

Later progress on the beyond-De Giorgi exact-rewrite branch only counts if a
bounded rewrite yields a smaller effective estimate on the fixed
`I_flux[phi]` term under the same cutoff protocol and solution class, after
all projection, Calderon-Zygmund, and commutator costs are paid.

This means the branch's gain currency is fixed in advance:
`coefficient shrinkage in the fixed localized LEI balance`.

This screen rules out three forms of fake progress:

- a prettier identity before localization
- derivative-count or exponent-improvement language that never returns to a
  smaller LEI cutoff-flux coefficient
- a gain only on a Lamb-vector side calculation with no LEI closure
  improvement
- a switch to a different architecture such as stretching localization
