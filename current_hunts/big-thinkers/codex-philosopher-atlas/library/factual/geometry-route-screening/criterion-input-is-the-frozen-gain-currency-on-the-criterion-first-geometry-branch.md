# Criterion Input Is The Frozen Gain Currency On The Criterion-First Geometry Branch

Status: `INFERRED` Step-015 currency freeze with `VERIFIED` support

Once the geometry branch freezes its theorem-facing comparator to the
Constantin-Fefferman-type direction-regularity family, the only honest
same-currency gain is `criterion input`.

That means Step-015 progress has to be measured against the comparator's own
criterion-scale direction burden:

- a candidate must supply an NS-specific insertion that is strictly cheaper
  than importing the static comparator's direction-control burden;
- and that cheaper input must still be witnessed through a smaller
  full-stretching contribution rather than through descriptive geometry alone.

So statements such as `direction coherence looks favorable on intense regions`
do not count as gain by themselves. They matter only if they weaken the
criterion input on the same theorem-facing ledger.
