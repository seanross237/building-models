# Fixed Package Makes Direction Coherence Informative But Dynamically Weak

Status: `VERIFIED` with `INFERRED` consequence

Under the Step-3 neutral Eulerian package
`Q_r(x_*, t_*) = B_r(x_*) x [t_* - r^2, t_*]` with intense set
`E_r(t) = {x in B_r(x_*) : |omega(x,t)| >= r^-2}`, vorticity-direction
coherence remains stretching-facing but does not become a strong standalone
dynamic route.

The useful part survives: direction coherence is still the comparator that
most directly talks to the stretching factor in the standard vorticity
amplitude story. But the same package also exposes the real burden. The local
corpus gives no repository-backed localized persistence bridge for coherence on
the moving intense set across the full `r^2` window, and any honest attempt to
keep that coherence localized naturally pulls in hidden `nabla xi`-type control
or threshold-boundary control.

So the comparator remains informative as a benchmark, but not dynamically
strong. Under the fixed package it tends to collapse either into prior-art
direction criteria or into a stronger derivative-control hypothesis that the
screen is not allowed to import for free.
