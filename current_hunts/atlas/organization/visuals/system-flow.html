Core exploration loop (repeats per exploration)
Sean
Defines the Mission and the Mission Validation Guide
Missionary
strategy builder
Reads: Mission, Mission History, Library Context

Outputs: Methodology
(objective, methodology, validation criteria, context)
Strategizer
Reads: Goal from Missionary, Strategizer History, Library Context, Validation Guide

Creates: Exploration Goal
Explorer
Executes the exploration goal.

Produces: Report
Disposable -- clean slate each run.
Scribe
Distills the Explorer's Report into:

1. Factual findings + Meta-knowledge
Filed by Library Curator
2. Structured report
Appended to Strategizer History
20-Loop Check-in
or strategy completion
Results flow back to Missionary.
Missionary reviews progress and decides next strategy direction.
Library Receptionist
Retrieves relevant library context on request.

Serves: Missionary, Strategizer
Library Curator
Files new findings into libraries.
Logs overwrites to Changelog.
Factual Library
Domain knowledge organized by topic
Meta Library
Agent capabilities, effective patterns
Changelog
Tracks all library overwrites
