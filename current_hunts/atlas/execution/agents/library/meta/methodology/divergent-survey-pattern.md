## Lesson

For Phase 1 divergent explorations, use the four-part survey pattern: (a) comprehensive survey, (b) deep dive on top 2-3 candidates, (c) synthesis attempt, and (d) honest novelty assessment. This combination produces high-quality results in a single exploration. Do NOT add construction on top -- that needs its own exploration.

## Evidence

- **strategy-001 exploration 001** — Survey + deep dive + synthesis + novelty assessment in one exploration. Explorer handled all parts well. Specifying "honestly assess novelty" and "does it actually derive QM or smuggle it in?" produced high-quality critical analysis. The explorer didn't oversell.
- **strategy-001 exploration 002** — Same pattern confirmed. Asking "is this genuinely new or repackaging?" got a nuanced answer distinguishing genuine novelty from repackaging.

## When to apply

The start of any new research direction. When you need to map a landscape before committing to a specific approach. The key constraint: ask for honesty explicitly ("honestly assess novelty," "is this genuinely new?") or the explorer will default to enthusiasm.
