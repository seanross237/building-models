# Self-Consistency of eta_h on a d_s = 2 Background

The graviton fluctuation anomalous dimension eta_h ~ +1 (computed by Bonanno, Denz, Pawlowski, Reichert on a flat Euclidean background) was checked for self-consistency when the background geometry has spectral dimension d_s = 2, as required by BMEG theory.

## Central Result

The self-consistency check is **conditionally passed**. The value eta_h ~ +1 is likely to shift quantitatively on a d_s = 2 background, but the *sign* eta_h > 0 is robust on structural grounds.

## Scenario Analysis

| Scenario | Description | Probability |
|----------|-------------|-------------|
| A | eta_h ~ +1 unchanged | 25% |
| B | 0 < eta_h != 1, sign preserved | 60% |
| C | eta_h changes sign or pathological | 15% |

Scenario B (most likely) means BMEG predictions shift quantitatively but the theory's structure remains intact.

## The Self-Consistency Loop

There is an elegant self-consistency argument:

1. Asymptotic safety *predicts* d_s = 2 in the UV (from eta_N = 2 - d)
2. BMEG *assumes* d_s = 2 as the background
3. On this background, eta_h > 0 is still generated by the graviton self-interaction
4. The fluctuation propagator ~ 1/p^{2-eta_h} is distinct from the background propagator ~ 1/p^4
5. The bi-metric structure is self-consistent: two different anomalous dimensions for two different physical sectors

The d_s = 2 background is not a foreign input -- it is what asymptotic safety itself predicts. Computing on this background is *more* self-consistent than computing on a flat (d_s = 4) background, which is what standard AS does despite predicting d_s = 2.

## Five Supporting Pillars

1. The exact relation eta_N = 2 - d at the NGFP means d_s = 2 is a self-consistent *output* of asymptotic safety regardless of background input
2. The graviton antiscreening mechanism generating eta_h > 0 is structural (spin-2 self-interaction), not background-dependent
3. The bi-metric framework separates eta_bar from eta_h -- they are governed by different physics
4. An explicit one-loop power-counting argument shows 1/p^4 background propagator *improves* UV convergence
5. The 2+epsilon expansion provides a perturbative cross-check: eta_h ~ O(epsilon) > 0 near d = 2

## Key Open Question

No explicit computation of eta_h on a d_s = 2 background exists yet. The highest-priority follow-up is computing this, using either:
- A modified FRG flow with non-standard heat kernel
- A self-consistent bi-metric computation where both sectors reach fixed points simultaneously
- A 2+epsilon approach with scale-dependent epsilon

## Critical Gap

The deeper issue remains: how does 4D macroscopic spacetime emerge from the d_s = 2 UV structure? This is not addressed by the self-consistency check and remains the critical gap in BMEG.
