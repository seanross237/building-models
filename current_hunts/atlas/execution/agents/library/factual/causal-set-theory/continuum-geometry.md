# Recovering Continuum Geometry: The Hauptvermutung

A central question in causal set theory is whether the discrete causal set uniquely determines the continuum spacetime it approximates. This is formalized as the Hauptvermutung ("fundamental conjecture").

## Statement of the Hauptvermutung

The Hauptvermutung asserts: if a causal set C can be faithfully embedded into two distinct Lorentzian manifolds (M1, g1) and (M2, g2), then those manifolds must be approximately isometric at scales much larger than the Planck scale. In other words, the causal set uniquely determines the large-scale geometry.

## Recent Results (Dribus, 2025)

A 2025 paper (arXiv:2503.01719) distinguishes two precise formulations and proves:

- **Theorem 1 (Countable Hauptvermutung -- TRUE):** When using countable (rather than finite) causal sets, the mapping from spacetimes to causal order relations is injective. Each causal set determines at most one spacetime up to isometry. This means general relativity can in principle be reformulated as a theory on P(N x N).

- **Theorem 2 (Finite Hauptvermutung -- FALSE):** For finite causal sets under the standard Gromov-Hausdorff distance d^-, there exist spacetimes that are arbitrarily far apart yet produce nearly identical probability distributions when sampled with finitely many points.

- **Theorem 4 (Metric-Dependent Recovery):** Under "Planck-scale uniformness" assumptions and using an alternative distance metric d_x, the finite Hauptvermutung becomes true. The choice of distance metric on spacetime fundamentally affects reconstructibility.

## Recovering Topology

Major and collaborators showed that the homology of a globally hyperbolic spacetime can be recovered from a faithful causal set embedding using "thickened antichains" -- a construction that provides a purely order-theoretic approximation to spatial topology at sufficiently high sprinkling density.

## Recovering Dimension

The Myrheim-Meyer dimension estimator allows one to estimate the spacetime dimension from the causal set alone. For a causal set faithfully embedded in d-dimensional Minkowski spacetime, the fraction of causally related pairs among all pairs in a causal interval determines d.

## Status

While a large body of evidence supports the Hauptvermutung, no complete proof exists for the finite case under general conditions. The 2025 results clarify that the conjecture's truth depends sensitively on the choice of distance metric used to compare spacetimes.
