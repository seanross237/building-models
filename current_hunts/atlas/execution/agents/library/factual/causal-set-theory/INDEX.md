# Causal Set Theory

Spacetime as a fundamentally discrete, locally finite partial order. Founded by Bombelli, Lee, Meyer, and Sorkin (1987). Only approach combining discreteness with Lorentz invariance; produced the only successful advance prediction in quantum gravity (cosmological constant).
