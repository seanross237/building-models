# Horava-Lifshitz Gravity

Quantum gravity via anisotropic scaling between space and time (z = 3), breaking Lorentz invariance to achieve d_s = 2 without ghosts. The U(1) extension resolves the scalar mode pathology. Constrained by super-Planckian LIV bounds from GRB 221009A.
