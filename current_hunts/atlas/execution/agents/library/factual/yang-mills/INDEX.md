# Yang-Mills Existence & Mass Gap — INDEX

Constructive quantum field theory approaches to the Yang-Mills Millennium Prize Problem: Balaban's renormalization group program, the precise gap structure toward a complete proof, and modern developments. 7 findings.

## Core Results

- **balaban-uv-stability.md** — Balaban's 14-paper RG program (1983-1989): proves UV stability of 4D lattice pure Yang-Mills in finite volume via four-phase construction (foundational tools → 3D proof → small-field 4D RG → large-field analysis). Precise theorem: partition function bounded uniformly as lattice spacing → 0. Also completes 3D UV stability (superrenormalizable case).

- **gap-structure-overview.md** — Two-tier gap structure toward the Millennium Problem: Tier 1 (potentially tractable with known methods) = observable control on T⁴ + uniqueness of continuum limit; Tier 2 (fundamental obstruction, needs new ideas) = mass gap + infinite volume limit. 12-step reconstruction chain with status. Four specific gaps with mathematical formulations and tractability assessments.

## Modern Developments (Post-Balaban)

- **dimock-expository-program.md** — Dimock's (2011-2022) revisitation of Balaban's methods: three-part expository series on φ⁴₃, UV stability for QED in 3D (2022), non-linear sigma model extensions. Most active ongoing effort to extend Balaban's RG.

- **chatterjee-probabilistic-program.md** — Chatterjee's (2018-2024) probabilistic approach: reformulation for probabilists, Wilson loop expectations for finite groups, strong mass gap ⟹ area law (confinement), SU(2) YMH scaling limit (2024, but Gaussian/trivial).

- **stochastic-quantization-chandra-hairer.md** — Chandra-Hairer-Chevyrev-Shen (2022-2024): stochastic quantization of 3D Yang-Mills-Higgs via regularity structures. Fundamentally different from Balaban — constructs dynamics rather than measure. Promising but incomplete.

- **other-modern-approaches.md** — Alternative UV stability proofs and continuum approaches: Faria da Veiga-O'Carroll (2019, simpler lattice proof), Magnen-Rivasseau-Sénéor (1993, continuum with IR cutoff), Charalambous-Gross YM heat flow regularization, Federbush phase cell approach.

## Reference Constructions

- **completed-gauge-constructions.md** — The only completed constructions satisfying OS axioms: Brydges-Fröhlich-Seiler 2D abelian Higgs (with mass gap by BBIJ 1984), King's 3D Higgs continuum limit, Gross/Driver abelian YM. These show the program CAN work for simpler models.
