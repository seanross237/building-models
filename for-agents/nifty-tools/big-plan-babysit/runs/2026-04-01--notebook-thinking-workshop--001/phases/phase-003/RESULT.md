# Result

## Session Spine With Timing

- `0-5 min`: set the decision question and write the workshop aim
- `5-10 min`: capture raw notes and observations
- `10-20 min`: sort notes into observations, implications, and open questions
- `20-35 min`: convert patterns into draft decisions and tradeoffs
- `35-50 min`: write the decision memo in full
- `50-60 min`: read it aloud, mark disagreements, and agree the next step

## Activities And Outputs

- set the question -> output: one agreed problem statement
- capture notes -> output: raw note list
- sort notes -> output: grouped evidence map
- shape decisions -> output: decision statements with reasons
- draft memo -> output: complete draft decision memo
- review and close -> output: signed-off memo or unresolved items list

## Final Memo Structure

- title
- decision question
- context
- key observations
- what we think they mean
- decision or preferred direction
- tradeoffs and risks
- open questions
- next step

## Why This Is Smallest Viable

It uses only the allowed materials, stays inside the time budget, and still produces the promised memo artifact.
