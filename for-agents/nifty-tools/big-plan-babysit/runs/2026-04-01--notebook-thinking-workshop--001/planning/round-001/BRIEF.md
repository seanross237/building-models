# Planning Brief

## Objective

Create a small but clean plan for producing a concise internal brief for a fictional workshop called `Notebook Thinking`.

The final brief must include:

- target audience
- workshop promise
- three-session outline
- one-paragraph pitch

## Planning Requirements

- produce a top-level plan with clear phases
- prefer `2-3` phases
- keep the plan proportional to the toy goal
- define acceptance criteria for each phase
- avoid unnecessary research or external dependencies

## Execution Environment

- a supervisor agent will execute one phase at a time
- worker subagents can be used for individual phases
- all artifacts must be written into this run folder

## Evaluation Priorities

- coherence
- simplicity
- good sequencing
- low overhead
- ability to evaluate phase completion cleanly
