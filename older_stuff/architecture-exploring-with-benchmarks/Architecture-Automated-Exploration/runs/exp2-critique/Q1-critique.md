# Q1 — FTIR Tardigrade Hydrogel Plan Review

## VERDICT: APPROVED

## ISSUES: None fatal. Two minor observations noted below for completeness.

### Detailed Review

**Peak assignments (Step 1) — CORRECT**
- 1645 cm⁻¹ = random coil / disordered: Standard assignment (~1640-1648 range).
- 1652 cm⁻¹ = alpha helix: Standard assignment (~1650-1658 range).
- 1618 cm⁻¹ = intermolecular beta sheet: Standard assignment (~1615-1625 range). This low-frequency component is characteristic of aggregated or intermolecular beta sheets, consistent with a gelation process.
- 1680 cm⁻¹ = antiparallel beta sheet high-frequency component: Standard assignment (~1680-1690 range). The fact that 1618 and 1680 disappear together on heating confirms they are the paired components of antiparallel beta sheet structure.

**Starting state (Step 3) — CORRECT**
The question explicitly states the proteins are "initially disordered but rapidly assume order at high enough concentration." The plan correctly identifies 1645 growth on heating as corroboration (thermal melting restores disorder).

**Heating behavior (Step 4) — CORRECT**
1618 + 1680 disappear together on heating = antiparallel beta sheet melting. 1645 grows = disorder returns. This is textbook thermal denaturation of beta sheet structure. Logic is sound.

**Concentration titration (Step 2) — CORRECT**
Dual increase of 1652 (helix) and 1618 (beta sheet) with concentration means BOTH secondary structures form during the concentration-driven gelation. This is the critical datum that eliminates single-structure answers (F, D, etc.).

**Answer selection (I) — CORRECT**
Scanning all options:
- A (helix -> disordered): Wrong direction.
- B (disordered -> helix): Missing beta sheet formation.
- C (coiled-coils): Would not produce 1618/1680 beta sheet signatures.
- D (helix -> beta sheet): Starting state is disordered, not helical.
- E (beta sheet -> helix): Starting state is disordered, not beta sheet.
- F (disordered -> beta sheet): Missing the 1652 (helix) increase in concentration titration.
- G (parallel -> antiparallel beta): Starting state is disordered, and this wouldn't explain 1652 growth.
- H (beta sheet -> disordered): Wrong direction; beta sheets FORM during gelation.
- **I (disordered -> beta sheets and alpha helices): MATCHES all data.**

### Minor Observations (non-fatal)

1. **Shoulder vs. peak subtlety:** The 1652 cm⁻¹ is labeled (sh) in the initial spectrum, meaning trace alpha helix content may already exist before gelation. The plan implicitly treats the starting state as purely disordered. This does not affect the answer — the concentration titration shows 1652 *increasing*, so helix is clearly forming regardless of whether a trace amount pre-exists.

2. **Potential spectral overlap concern:** One could worry that the 1652 increase is an artifact of the broad 1645 peak's shoulder shifting. However, the concentration titration shows 1652 and 1618 increasing in tandem, while the heating experiment shows 1645 increasing as 1618/1680 decrease. These are opposite trends at 1618, confirming that the 1652 concentration-dependent growth is genuine helix formation tracking with beta sheet formation, not a broadening artifact of the disorder peak.

Neither observation changes the conclusion.

(2) Concentration titration shows DUAL increase of 1652+1618 → both helix and sheet form during gelation. (3) Starting state is disordered (stated in problem + 1645 grows on heating = disorder returns). (4) Heating melts 1618+1680 together = beta sheets disassemble, confirming they formed during gelation. Only answer matching "disordered → both beta sheet AND alpha helix" is **I**.
